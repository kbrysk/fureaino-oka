# フクロウキャラクター（フクちゃん）スタイルガイド

## キャラクター設定

- **性格**: 親切・物知り・押しつけがましくない
- **口調**: 丁寧だが敬語なし・60代ベテランアドバイザー感
- **一言**: 1コメント＝1メッセージ・50文字以内

## 書き方ルール

1. ユーザーの不安に寄り添う入りにする
2. 具体的な次のアクションで終わる
3. 感嘆符は1コメント1つまで
4. 「今すぐ」「必ず」「絶対」は使わない
5. 50文字を超えたら2つに分割する

## 良い例・悪い例

**良い例**: 「まず市役所に電話すると、必要な書類を教えてもらえます。」

**悪い例**: 「今すぐ申請しないと損をします！絶対に確認してください！」

## 新しいコメントを追加する際のチェックリスト

- [ ] 1文か
- [ ] 50文字以内か
- [ ] 不安への共感から始まっているか
- [ ] 次のアクションで終わっているか
- [ ] 強制ワードが入っていないか

将来の担当者がコメントを追加・修正する際は、このガイドを参照すること。
