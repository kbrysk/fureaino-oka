# 村上様向け「監修者の見せ方」確認デモ

このフォルダは、村上充恵 様に「総合監修者として、実際のサイト上でどう見えるか／導線はどうなるか」を**クリックして確認していただく**ためのデモ一式です。本番サイトには一切影響しません。

## 何が見られるか
`murakami-supervisor-demo.html` を開くと、上部タブで次の4画面を回遊できます。
1. **監修者プロフィール** … 恒久ページのイメージ（資格・経歴・実体験・ル・リアン横浜への導線）
2. **監修記事の例** … 村上様のご専門領域の記事に「監修：村上充恵」が冒頭・末尾に入る様子
3. **専門外の記事の例** … 相続税など専門外の記事には**村上様の名前が出ない**（評判保護＝提案書§5の線引きを画面で実証）
4. **編集方針・監修体制** … 分野ごとの監修者一覧（透明性＝信頼の担保）

※判断用の「オプション（ご選択）」一覧は社内向けのため、村上様に見せるデモからは除外しています（選択肢は `docs/SUPERVISOR_DEMO_OPTIONS_AND_DECISIONS.md` に記載）。

常時表示のフッターに「総合監修（編集方針）：村上充恵」クレジットが入ります。

> **村上様はIT不慣れな前提**：いちばん確実なのは「PDFを添付して送る」方法です（下の見せ方0）。リンクで見せたい場合は見せ方AのワンタッチURLを使ってください。

## 見せ方0：PDFで渡す（最も確実・推奨）
`mockup/pdf/` に各セクションの配布用PDFを用意済み（日本語フォント埋め込み・検索除外）。村上様へメール／LINE添付でそのまま送れます。
- `01-profile.pdf` … 監修者プロフィール
- `02-supervised-article.pdf` … 監修記事の例（村上様クレジットあり）
- `03-non-supervised-article.pdf` … 専門外の記事の例（村上様の名前は出ない）
- `04-editorial-policy.pdf` … 編集方針
- `05-supervision-structure.pdf` … 監修体制
- `00-all-combined.pdf` … 上記5つを1ファイルにまとめた全部入り（1通で送りたい時に）

> 元データは `mockup/pdf-src/*.html`。文面を直したら `node`/Chrome で再生成できます（手順は下部メモ、または私に依頼）。

## 見せ方A：ワンタッチURLを作る（推奨・所要約30秒）
公開用に整えた `mockup/dist/`（中に `index.html`／検索除外noindex付き）を、無料の静的ホスティングにドラッグするだけで、クリックできるURLになります。

**A-1. Netlify Drop（アカウント不要で最速）**
1. ブラウザで https://app.netlify.com/drop を開く
2. エクスプローラーで `mockup/dist` フォルダを、ページ上にドラッグ＆ドロップ
3. 数秒で `https://○○○.netlify.app` のURLが発行される → これを村上様へ送る
   - URLを残したい場合は無料サインアップで「サイトを保持」。第三者には推測されにくいランダムURL。

**A-2. Vercel（すでにお使い・自社環境で）**
- ターミナルで `cd mockup/dist && npx vercel --prod`（初回は `vercel login` でメール認証）。クリーンな `https://○○○.vercel.app` が発行されます。本体リポジトリやデプロイには影響しません。
- ※私（Claude）が代わりに実行することも可能ですが、Vercel/Netlifyへのログイン認証はご本人操作が必要です。ご希望なら声をかけてください。

いずれも村上様側は**リンクをタップするだけ**。確認画面や警告は出ません。

## 見せ方B：ファイルを直接開く（最速・オフライン）
`murakami-supervisor-demo.html` を**ダブルクリック**でブラウザ表示。自分の確認用・対面で見せる用に。村上様に送る場合はダウンロードが必要なのでAを推奨。

## 見せ方C：GAS Webアプリ（非推奨・初回に警告画面）
`gas/Code.gs` の手順でURL化できますが、初回アクセス時に「このアプリはGoogleで確認されていません」という**警告画面が出るため、IT不慣れな村上様には不向き**。Aを推奨します。

## 見せ方D：本番に近い実画面で見せる（後日）
方向性が固まった後、ふれあいの丘の**実ページ**として実装します（`/supervisor/murakami` 等）。下準備として未配線の実Next.jsページ案を `app/supervisor/murakami/` に用意済み（本番フッターは未改変）。

> メモ: `mockup/dist/index.html` は `murakami-supervisor-demo.html` の公開用コピーです。デモを修正したら `cp mockup/murakami-supervisor-demo.html mockup/dist/index.html` で再コピーしてください（私に頼んでもOK）。
