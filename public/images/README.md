# サイトで使う画像

## owl-character.png

**フクロウのキャラクター画像**（ナビ・フッター・ガイドブック・記事などで使用）

- ファイル名: `owl-character.png`
- このフォルダに `owl-character.png` を置いてください。
- メガネとマフラーをしたふくろうのイラスト（推奨: 正方形に近い画像）

## hero-couple.jpg

**ヒーローセクション背景画像**（トップページのファーストビュー）

- ファイル名: `hero-couple.jpg`（PNG の場合は `app/page.tsx` の `hero-couple.jpg` を `hero-couple.png` に変更）
- このフォルダに、夫婦がテーブルで本を読む明るい光の入る画像を置いてください。
- 推奨: 横長（例: 1200×630px 以上）。光の筋が右側に入っている構図。

置いたら開発サーバーを再起動し、ブラウザをハードリロード（Ctrl+Shift+R / Cmd+Shift+R）してください。
