# SEO×生前整理 ディスカッション設計メモ

検索クエリごとの**ユーザーニーズ・心理的ニーズ・抵抗感**を整理し、コンテンツの肉付け方針を定義したメモ。

---

## 1. 実家 片付け 親 説得（共感・対話法）

| 観点 | 内容 |
|------|------|
| **検索意図** | 親を説得する「具体的な言い方・会話のコツ」が欲しい。説得に失敗した・こじれた経験がある人も多い。 |
| **心理的ニーズ** | 親を傷つけず、関係を悪化させずに進めたい。自分が「悪者」になりたくない。親の気持ちを理解したうえで動かしたい。 |
| **抵抗感** | 「説得」という言葉への罪悪感。親を追い詰めている感覚。何度言っても聞いてもらえない無力感。 |
| **コンテンツ方針** | 説得の「前に」親の心理（もったいない・体力衰え・アイデンティティ）を丁寧に解説。会話の「具体的な言い換え例」を豊富に。デス・クリーニングの「少しずつ・他者への思いやり」を根拠として提示。第三者・ツールの活用で対立構図を和らげる。FAQで「何度言っても聞かない」「親が怒る」への回答を明示。 |

## 2. 実家 片付け どこから（手順・ロードマップ）

| 観点 | 内容 |
|------|------|
| **検索意図** | 最初の一歩と、場所別の順番が欲しい。全体像が見えず不安。 |
| **心理的ニーズ** | 迷いなく始めたい。挫折したくない。達成感を得たい。 |
| **抵抗感** | 量の多さに圧倒されている。どこから手をつけていいかわからない不安。 |
| **コンテンツ方針** | 結論ファーストで「全体像の可視化→一箇所決定→実行」を明示。場所別の難易度・心理的ハードル・おすすめ順を表やリストで示す。「続けるコツ」で進捗可視化・時間区切り・診断ツールへの誘導。具体例（「今週は押入れだけ」）を多数。 |

## 3. 生前整理 やり方 リスト（手順・リスト）

| 観点 | 内容 |
|------|------|
| **検索意図** | やることを漏れなくリスト化したい。チェックリスト・テンプレが欲しい。 |
| **心理的ニーズ** | 漠然とした不安を減らしたい。進捗が見えると安心。 |
| **抵抗感** | 範囲が広く何から書けばいいかわからない。リスト作りで詰まる。 |
| **コンテンツ方針** | リストが有効な理由（可視化・進捗・共有）を心理面から説明。フェーズ分け（把握→優先順位→実行）を明確に。場所別・種類別のロードマップを詳細に（書類・衣類・思い出・デジタル）。無料チェックリストへのCTA。 |

## 4. 実家 片付け 業者 頼みたい（代行・外注）

| 観点 | 内容 |
|------|------|
| **検索意図** | 業者に頼んでいいか・頼むタイミング・選び方・費用感が知りたい。 |
| **心理的ニーズ** | 家族だけで抱え込まなくていいと安心したい。悪質業者に当たりたくない。 |
| **抵抗感** | 業者＝高額・押し売りのイメージ。親孝行を放棄しているような罪悪感。 |
| **コンテンツ方針** | 「プロに頼むのは親孝行」と肯定的に位置づけ。依頼の段階的パターン（一部だけ頼む等）を提示。見積もり内容・買取時の説明・断りやすさ・一括見積もりの重要性。消費者ホットライン・法テラス等の相談窓口を明記。 |

## 5. デジタル遺品 パスワード わからない（啓蒙・ツール）

| 観点 | 内容 |
|------|------|
| **検索意図** | パスワードがわからないと何が困るか、今から何をすればいいか知りたい。 |
| **心理的ニーズ** | 家族が将来困らないようにしたい。自分ごととして備えたい。 |
| **抵抗感** | 面倒・先延ばし。セキュリティへの不安。何を書いていいかわからない。 |
| **コンテンツ方針** | わからないとどうなるか（相続遅れ・サブスク継続・データ喪失）を具体的に。目録の項目例を網羅的に。パスワード管理アプリ・「残す・共有・消す」の整理。相談先（金融機関・士業）を明示。 |

## 6. 着物 処分 罪悪感（価値転換・供養）

| 観点 | 内容 |
|------|------|
| **検索意図** | 罪悪感を減らしたい。捨てない選択肢（買取・寄付・供養）が知りたい。 |
| **心理的ニーズ** | 形見を粗末にしたくない。祖母・母への思いを裏切りたくない。 |
| **抵抗感** | 処分＝裏切りという罪悪感。「もったいない」教育との葛藤。 |
| **コンテンツ方針** | 罪悪感が生まれる理由を肯定（形見・思い出・もったいない教育）。捨てない選択肢を優先（リユース・買取・寄付・供養）し、心の整え方（長く大切にした意味・少しずつ手放す）を丁寧に。買取・査定の安心ポイント（無料・断りやすさ・勧誘規制）。 |

## 7. 骨董品 価値 わからない（査定・鑑定）

| 観点 | 内容 |
|------|------|
| **検索意図** | 価値の見極め方・無料鑑定・安心して依頼する方法が知りたい。 |
| **心理的ニーズ** | 捨てて後悔したくない。高価なものなら適正に売りたい。 |

## 10. 生前整理 いつから（きっかけ・年齢）

| 観点 | 内容 |
|------|------|
| **検索意図** | いつから始めるのが適切か・年齢別・今からでも遅くないか知りたい。 |
| **心理的ニーズ** | 早すぎる・遅すぎるという不安を解消したい。今すぐできることが知りたい。 |
| **抵抗感** | 死を連想して避けたい。まだ早いという先延ばし。 |
| **コンテンツ方針** | 「早すぎることはない」「今からで大丈夫」を明確に。年齢・きっかけ別の目安。今すぐできる第一歩（リスト・1箇所・エンディングノート等）を具体的に。 |
