## 0. エグゼクティブサマリー（結論）

1. **1,000万円は「収益倍率」では年内に到達不能、「戦略売却」では到達可能圏。** 国内相場（ラッコM&A成約1,050件・検証済み）では収益ゼロサイトの平均成約は**13.9万円**。1,000万円には月利37万（仲介型27ヶ月倍率）〜55万（フリマ実勢17.8ヶ月）の**6ヶ月平均実証**が必要で、現状（累計収益¥5,000・月クリック2）からは物理的に間に合わない。
2. **一方、内部の代替コスト見積（DB構築500〜800万＋リードインフラ300〜500万）だけで1,000万円は正当化でき**、買い手候補は実在が検証できた（クラッソーネ＝自治体×解体データが戦略コア／バリュークリエーション＝2024年に小規模サイト2件を実買収／鎌倉新書・じげん＝買収窓口公開中）。海外でも「収益でなく資産を隣接事業者が買う」直接の先行事例（Cake→Foundation Partners 2024）が確認済み。
3. **最大の設計変更: B2Bデータライセンスを「代替案」から「主軸エンジン」に昇格する。** 国際航業が自治体補助金データ（約2,000件）を**API月額10〜30万円/Excel月額50万円**で現に販売している（検証済み）。当社の1,726市区町村・解体補助金DBは同型資産であり、**B2B契約はトラフィックを必要としない**＝現状の「クリック700〜1,400倍ギャップ」を回避できる唯一の収益経路。B2B契約1〜2件（月5〜15万円）が成立すれば「契約付き継続収益＋複製困難なDB」となり、戦略売却の価格根拠が収益面からも補強される。
4. **確実性についての誠実な認識**: 年内1,000万円クローズの成功確率は、施策完遂を前提に**おおむね30〜45%**（B2B契約獲得時は+10〜15pt）。「確実」は構造的に存在しない。本設計は (i)確率を最大化し (ii)失敗時の下振れを「2027H1に月利実績付きで再挑戦（期待値300万〜1,000万）」に限定する、**分散最小化設計**である。

## 2. 到達ロジック — 「二重エンジン＋下限防御」設計

### 2-1. なぜ1,000万円は届き得るのか（論理の連鎖・飛躍なし）

```
前提1: 買い手は「再構築コスト vs 買収価格」で意思決定する（build-vs-buy。海外事例・国内買収目的の文言で確認）
前提2: 当社DB＋ツール群の再構築コストは800〜1,300万円相当
       （DB: 1,726自治体×調査・構造化・検収＝500〜800万／ツール9本・リード導線: 300〜500万）
前提3: 戦略フィットが最大の買い手（クラッソーネ型）にとってDBは「自治体営業の即戦力資材」
       ＝単なるサイト買収でなく事業資材の取得（国際航業が同型データを月10〜30万で売れている＝市場価値の証明）
結論: 「1,000万円 ≦ 再構築コスト」の提示は、戦略買い手にとって合理的検討範囲に入る
```

**変数とボラティリティ**: この連鎖の弱点は「買い手が見つかる確率」のみ（価格論理は内部完結）。だから施策の大半を「買い手発見の確率を上げる」ことに割く——①打診母数の最大化（5カテゴリ×50社）②証拠の補強（B2B契約・初CV・順位カーブ）③DD摩擦の除去（クリーン数字・データルーム）。

|---|---|---|---|

### 2-3. やらないこと（検証結果により更新）

1. AdSense最適化（月30-60万PV必要＝論外）
2. 遺品整理アフィリのCTA主役起用（単価2.5-6千円×契約完了承認＝工数対効果が劣後）
3. フリマ型マーケプレへの「収益ゼロのまま」出品（平均13.9万円の値しか付かない。価格床を自ら毀損する）
4. 新規KW量産・pSEO拡大（HCS逆行）
5. 「検索流入の将来性」を売り文句の中心に置くこと（AI Overviews時代に買い手が割り引く。DB・ツール・リストで語る）
6. GA4総数の対外使用（ボット19%混入。GSC実数とクリーン数字のみ）

## 3. タイムスパン別アクションプラン

### 3-1. 長期（〜2027年6月: 12ヶ月視野の選択肢設計）

**目的: 「年内に売れなくても損しない」構造を作る。**

- 年内クローズ（確率30-45%）に失敗した場合、**2027H1に「運営18-24ヶ月・月利6-12ヶ月実証」のプロフィールで再打診**。サイトマ事例（月利15万+×複数年→500-1,200万円）に接続する
- B2Bデータ事業は売却の成否に関係なく育てる（Googleアルゴリズム・AI Overviewsの影響を受けない収益＝terminal valueの保険）

## 6. 本設計の正直な限界（論点の透明化）

1. **「確実な1,000万円」は存在しない。** 本設計が保証するのは (i)価格論理の防御可能性 (ii)買い手発見確率の最大化 (iii)失敗時の下限（2027年に300〜600万＋再挑戦権）の3点。
2. B2B価格（月5〜15万）は国際航業（月10〜30万）からの**類推**であり、解体・空き家ニッチでの支払意思は未検証。7〜8月の営業20社が最初の実地検証になる。
3. 買い手5社の「買収意欲」は公開情報からの推定。クラッソーネの戦略フィットは検証済みだが、同社が「作る」より「買う」を選ぶかは接触するまで不明。
4. 成功確率30〜45%は主観的事前確率。8月末・9月末ゲートの実測で更新する（ベイズ更新を月次で明記）。
