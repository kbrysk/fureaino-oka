# データレポートの「発見される設計」＋都道府県別47面＋図版資産（2026-06）

> 目的: メディアに引用してもらう前提＝「①引用しやすい ②ためらわせない ③引用したくなる」を満たしつつ、
> そもそも検索で見つけてもらうための**発見性（ディスカバビリティ／SEO）**を設計・実装する。

---

## 1. なぜ「発見されない」と引用されないのか（問題の起点）

メディアが引用するには、まず**検索や巡回でそのデータに辿り着く**必要がある。精査の結果、致命的なギャップが判明：

- 🔴 **データレポートがサイトマップに1本も載っていなかった**（`/sitemap.xml` の子 `/sitemap/main.xml` に `/data/*` が不在）。＝Googleに「存在を申告」していなかった。
- 🔴 **データ室（ハブ）が無く**、内部リンクが集約されず、クロール経路が細かった。
- 🟠 **OG画像が無く**、SNS/メディアで共有されても地味で拾われにくかった。
- 🟠 **引用面が全国1枚のみ**で、地方紙・自治体系メディア（「○○県の補助金」を探す層）に当たらなかった。

---

## 2. 実装した「発見される設計」

### A. クロール・インデックスの土台
- **サイトマップ登録**: `/data`・`/data/akiya-hojokin-ranking`・`/data/seizen-seiri-trends`＋**都道府県別47面**を `app/sitemap.ts`（正式 `/sitemap.xml`）と route-handler 版の両方に追加。
- **データ室ハブ `/data` を新設**: 全レポート＋47県への内部リンクを集約（リンクエクイティの親）。`CollectionPage`＋`BreadcrumbList` 構造化。
- **内部リンクの面を増設**:
  - 全国レポート → 47県ページ（「都道府県別の詳細データ」一覧）
  - 47県ページ → 全国・他県・該当県の `/area`（相互リンク網）
  - `/area/[県]` ハブ → 該当県のデータページ（トピック完全一致リンク）
  - `/akiya` ピラー → 全国レポート（既存）

### B. クリックされる・拾われる設計
- **OG画像＝インフォグラフィック**を全国/県別レポートに設定（SNS共有・メディア転載で視認性UP）。
- **検索クエリ最適化**: title/description を「○○県 空き家 解体 補助金 一覧・ランキング」等の実検索語に合わせ、母数・中央値を要約に含めCTRを上げる。

### C. AI検索・データ検索での発見（GEO）
- **Dataset 構造化＋`distribution`（CSV/JSON）**で Google Dataset Search に登録。県別ページにも `spatialCoverage=県名` の Dataset を付与。
- 冒頭に**断定的な要点（確認できた割合・中央値）**を置き、AI Overview/AI検索の引用を狙う。

---

## 3. 都道府県別レポート47面（地方メディアの引用面）

- ルート: `/data/akiya-hojokin-ranking/[prefecture]`（`generateStaticParams` で47面を静的生成）。
- 各県の**実統計で裏付け**（薄い量産にしない）:
  - 県内で解体補助金を**確認できた**数・割合（「確認できた」表現で不在を断定しない）
  - 県内の上限額**中央値/平均/最高**、**市区町村別ランキング**（各市の `/area/.../subsidy` へリンク）
  - **全国比較**（県の割合 vs 全国、中央値、平均額の全国順位）
- **引用キット（県スコープ）**: 引用文3形式・埋め込みカード（地域名入り）・オープンデータDL。
- 誠実性・中立性・検証性（公式URL付き）・訂正窓口を全国版と統一。

> 効果: 「○○県 空き家 補助金」系の検索面を47枚に拡張し、**地方紙・自治体・地域ブロガーの引用面**を確保。各ページがDataset＋引用キットを持つため、拾われればそのまま被リンク化する。

## 4. 図版資産（インフォグラフィック）

- `public/opendata/akiya-hojokin-infographic.{svg,png}`（1200×630・OG比）を `npm run data:opendata` で**ローカル生成（日本語フォントあり）→コミット**。
- 内容: 「48.9%の自治体で解体補助金を確認」＋バー＋中央値¥50万＋カバー率99.1%＋出典/CC BY/URL。
- 用途:
  - **引用キットに「画像で引用」**（`<a><img></a>`＝相手ページに実リンク＋図版）を追加。
  - **OG画像**として全国/県別に設定（SNS拾われ向上）。
  - 四半期データドロップで自動再生成。

## 5. 「引用される3条件」への対応マップ

| 条件 | 実装 |
|---|---|
| **引用しやすい** | 引用文3形式コピー・URLコピー・埋め込みカード・**画像で引用**・CSV/JSON（公式URL付き） |
| **ためらわせない** | 「確認できた」表現で不在を断定しない・母数明示（全国1,741の99%）・中央値主役・外れ値注記・商用CTA分離・訂正窓口・CC BY |
| **引用したくなる** | ほぼ全数調査（99.1%）の希少性・断定統計・図版（インフォグラフィック）・県別の"自分ごと"データ |
| **発見される** | sitemap登録・/data ハブ・内部リンク網・OG画像・Dataset Search・検索語最適化・47面の面展開 |
