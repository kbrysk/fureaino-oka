# 現在地の冷徹な監査と、年内3,000万円エグジットへの逆算（2026-06）

> 入力データ: GA4（2026-04-01〜06-04）／Search Console（過去3か月・web）／A8.net（2026-01〜06 月次）。
> 4視点（SEOプロ／Googleアルゴリズム最高責任者／年内3,000万エグジット起業家／ノンペイド・メディアでExitするオーナー）で独立評価→統合。
> 結論を先に: **現状の数値のままでは「年内3,000万円の"収益マルチプル型"エグジット"は不可能**。唯一の現実解は**「データ資産＋自動化＋ページ在庫」を戦略的買い手に売る"アセット型"エグジット**であり、そのために今すぐ潰すべき"3つの致命的ギャップ"がある。

## 0. エグゼクティブ・サマリー（残酷な現在地）

数字は「右肩上がりに見える」が、3つを突き合わせると**ファネルが全段で壊れている**：

| 段 | 数字 | 実態 |
|---|---|---|
| 露出 | GSC 表示 約11,300/3か月・**平均掲載順位 56位** | 大量のページが**4〜10ページ目**。Googleが「価値が低い」と判断して上位に出していない |
| 流入（検索） | GSC **クリック 約39/3か月（CTR 0.36%）** | Google検索からの実クリックは**月13件**。pSEOページは"出るが押されない" |
| 流入（GA4） | GA4 Organic Search **1,034セッション** | **GSCの約26〜40倍**。この乖離＝後述の最重要リスク |
| 収益 | A8 **クリック 471(4月)/276(5月)→CV 1件・¥5,000のみ** | 集客はあるが**1件しか換金できていない**（CVR≒0.2%）。導線の意図不一致 |

→ **「順位56位・実クリック月13件・収益累計¥5,000・GA4とGSCが40倍乖離」**。これが買い手が最初に見る現実。年内3,000万円の収益マルチプル（≒月70〜100万円の利益が必要）には**約150〜200倍**足りない。

## 1. 致命的ギャップ①：GA4「Organic 1,034」 vs GSC「クリック39」＝最大40倍の乖離

**事実**: GA4 Organic Search は 65日で 1,034セッション（≒16/日）。GSCのGoogleクリックは3か月で約39（≒0.4/日）。Yahoo!JapanやBingを足しても説明できない（日本でYahooはGoogleの十数%規模）。

**意味（最重要）**: この乖離は**M&Aデューデリの一発アウト項目**。買い手は必ずGA4×GSCを突き合わせる。「自然検索が実クリックの40倍」＝**ボット/無効トラフィック/誤計測**を疑い、**ディール破談か評価額暴落**になる。
- もし乖離が"無効トラフィック"なら、4月のA8 471クリック→CV0件も同根（人間でないクリック→換金ゼロ）で筋が通る。
- もし"実在の非Google流入"なら、それはそれで**GSCで測れていない収益チャネルの正体を突き止める**価値がある。

**今すぐやること（P0）**:
1. GA4で **Organic の参照元（session_source/medium）・地域・デバイス・直帰/エンゲージ秒の分布**を確認。`search.yahoo.co.jp` `bing` 等の内訳と、`(not set)`/不自然な国・1秒未満セッションの塊を洗う。
2. **GA4のbot除外・内部トラフィック除外・データフィルタ**を点検（自社ツール/監視/Lighthouse/Indexing/クローラの自己流入）。
3. **GSCプロパティの整合**（URLプレフィックス vs ドメインプロパティ、www/非www）を確認し、本当にGoogleクリックが月13件なのかを確定。
4. 結論が出るまで、**事業計画・買い手向け資料では自然流入を"GSC実クリック水準"で保守的に置く**（盛らない＝DD耐性）。

## 2. 致命的ギャップ②：露出は出るのに順位56位＝Googleが「低品質」と評価している

**事実**: 表示11,300・順位56・CTR0.36%。PCは表示9,852で順位55.8・CTR0.14%（ほぼ死に枠）、モバイルは表示1,371・順位46.8・CTR1.82%。クエリは**「遺品整理 ◯◯市」「空き家処分 ◯◯市」「解体費用 ◯◯市」「◯◯市 空き家補助金」「リフォーム補助金 ◯◯市」「粗大ゴミ ◯◯」**等の**超ローカル**が大量。多くが**40〜99位**。

**意味**: これは典型的な**「スケールド・コンテンツ／薄いpSEOがインデックスはされるが順位は抑制される」**状態。"表示されるのに押されない"＝Googleアルゴリズムが「このページ群は競合に対して価値が低い」と言っている。3,500ページ規模に対しクリック39は、**ページ在庫が資産でなく"品質負債"になりかけている**サイン（HCS的な全体評価の足枷リスク）。

**ただし朗報**: クエリを精査すると、**補助金/固定資産税/制度系の超ロングテールは上位化が始まっている**：
- 「実家じまい 大府市」1位、「実家じまい 清須市」1.7位、「◯◯町空き家補助金/解体補助金」5〜15位が多数、「固定資産税 富士宮市」16位で**CTR100%**、「田辺市 固定資産税」18位で1クリック。
- 一方「遺品整理/空き家整理/処分 ◯◯市」は40〜90位（=実サービス業者がいる商用枠で勝てない／しかも換金しづらい）。

→ **勝ち筋は「補助金・税金・制度」クラスタ（独自DBが効く・上位化が始まっている）。負け筋は「遺品整理・処分の超ローカル商用」（勝てず・換金も弱い）。**

## 3. 致命的ギャップ③：集客は換金できていない（意図不一致）

**事実**: A8は4月クリック471→CV1件（¥5,000・確定100%）、5月276→0、6月9→0。累計収益**¥5,000**。
**意味**: **不動産査定（ノムコム）**は"空き家を所有し売る意思のある人"にしか刺さらない。だが実際に集めているのは**「遺品整理/粗大ゴミ/処分」の情報・サービス探索者**で、**査定には意図が合わない**。だから471クリックでも1件。
- 加えて、ギャップ①が"無効トラフィック"なら、そもそも換金不能。

**処方**: **ページの意図に合わせて出口を出し分ける**。
- 空き家・解体・補助金・固定資産税ページ → **不動産査定**（意図一致）＋解体見積もり系。
- 遺品整理・不用品・粗大ゴミページ → **遺品整理/不用品回収のリードジェン or 専用ASP**（査定を出さない）。
- 終活・生前整理の情報ページ → **無料PDF/メール（Zero-Party Data）獲得**に徹し、後追いで査定/相談へ。

## 4. 4視点 独立レーティング＋優先課題

### 視点① SEOプロフェッショナル — **3.5 / 10**
- 診断: 3,500ページ・表示11k・順位56・クリック39。露出はあるが順位が壊滅。pSEOの薄さ＋DR21（権威不足）で上位化しない。補助金/制度系のみ上位化の芽。
- 優先:
  1. **勝てるクラスタに集中**（補助金・固定資産税・制度・実家じまい）。負けクラスタ（遺品整理処分の超ローカル）への投資を止める。
  2. **"順位8〜25位＋表示あり"のページをTOP5へ**押し上げる（タイトル/メタ・内部リンク・データページ連携・少数の被リンク）。CTRはモバイル偏重なのでモバイルSERP最適化。
  3. **被リンク/権威**（DR21が"出るのに上がらない"主因）。データレポートの引用獲得を実行。
  4. **GA4×GSCの乖離を解消**（計測が信用できないと最適化判断がぶれる）。

### 視点② Googleアルゴリズム最高責任者 — **3 / 10（品質負債リスクあり）**
- 診断: 大量テンプレページ＋100記事一気投入＝スケールド・コンテンツの典型。インデックスはするが順位抑制＝アルゴリズムの「低価値」判定。サイト全体の品質評価を下げる火種。
- 優先:
  1. **品質の集約**: 順位が付かない薄い量産ページは"資産"でなく"負債"。独自データで実体化できるものは強化、できないものは**統合 or noindex**でクロール/評価を集中（※従来は保留方針だったが、データが"順位ゼロ"を示した今は再検討に値する）。
  2. **新規の薄いページ追加を止める**（補助金データ47面は実データなのでOK。これ以上のpSEO面拡大はリスク）。
  3. **E-E-A-T・独自性のシグナルを少数の勝てるページに集中**（データレポート・監修・データの堀）。
  4. **権威（被リンク・指名検索）**を上げ、ドメインの信頼を底上げ。

### 視点③ 年内3,000万円エグジット起業家 — **エグジット準備度 2.5 / 10**
- 冷徹な算数: 収益累計¥5,000・CV1件・CVR≒0。**収益マルチプル型で3,000万円＝月70〜100万円利益が必要**。年内にこの基盤からは届かない。
- 唯一の道: **戦略的買い手へのアセット売却**（1,726自治体DB＋自動化パイプライン＋3,500ページ在庫＋ドメイン＋ブランド＝"顧客獲得チャネル/データ"として評価される）。
- 優先:
  1. **GA4×GSCを今すぐクリーンに**（DDの一発アウトを除去。盛らない数字＝交渉力）。
  2. **意図一致で収益トレンドを作る**（小さくても"伸びるグラフ"。査定の出し分け＋リードジェン）。
  3. **アセットを売り物に整える**（DB・自動化・ページ在庫・（検証済み）トラフィック・収益トレンドの資料化）＋**戦略的買い手リスト**（解体ポータル/遺品整理チェーン/不動産/終活事業/自治体テック）。
  4. Q3に"伸びる×独自データ×自動化"の物語を作り、**Q4に売りに出す**。

### 視点④ ノンペイド・メディアでExitするオーナー — **資産構築 5／成果 2.5（総合 3.5/10）**
- 診断: ノンペイドのエンジン（pSEO＋記事＋データ＋自動化）は組み上がったが、**出力（クリック・収益）が出ていない**。DR21・被リンクゼロ・指名検索ゼロでフライホイールが回り始めていない。
- 優先:
  1. **一点突破**: 遺品整理/空き家/解体/生前整理/終活に分散せず、**「空き家×補助金×税金」一本に絞って支配**（=ランクする）。
  2. **表示→クリック転換**: 11k表示の中の"勝てる位置"を取りに行く。
  3. **実需に合わせた換金**（査定の出し分け＋遺品整理リードジェン）。
  4. **ブランド＝指名検索ゼロを脱却**（データPRで言及・被リンク・指名を作る）。

## 5. 統合ディスカッション → 年内3,000万円エグジットの"唯一の現実解"

4視点は一点に収束する：

> **現状は「露出はあるが順位が低い（Google評価が低い）／GA4が実態の40倍に見える（DDリスク）／集客が換金できない（意図不一致）」。この状態で収益マルチプル3,000万円は不可能。**
> **唯一の現実解は、(a) トラフィックの実在をクリーンに証明し、(b) 意図一致で小さくとも伸びる収益を作り、(c) 権威を上げて"勝てる1クラスタ"を上位化し、(d) 1,726自治体DB＋自動化＋ページ在庫＋ドメインを"戦略的買い手の顧客獲得アセット"としてパッケージして売ること。**

エグジット形態の比較：
| 形態 | 必要条件 | 年内可否 |
|---|---|---|
| 収益マルチプル（30〜40x月利益） | 月70〜100万円利益 | ❌ 不可能 |
| **戦略的アセット売却** | DB/自動化/在庫/ドメイン/（検証済み）トラフィックの"獲得チャネル価値" | ⭕ **唯一の現実解** |
| トラフィック×リードジェンの事業売却 | 実クリックが今の10〜50倍 | △ 半年では厳しい |

## 6. 優先施策ロードマップ

### P0（今週）— DDの地雷除去＋換金の整合
1. **GA4×GSC 乖離の解明**（参照元内訳・bot/内部除外・プロパティ整合）。結論を1枚にまとめ、以後の数字は保守的に。
2. **出口の意図出し分け**: 空き家/補助金/税金=査定、遺品整理/不用品=リードジェン、終活=PDF/メール。CTAクリックをGA4イベントで計測。
3. **勝てるページのTOP5化**: GSCで"順位8〜25×表示あり"を抽出→タイトル/内部リンク/データ連携で押し上げ。

### P1（今月）— 品質集約＋権威＋収益トレンド
4. **品質トリアージ**: 順位が付かない薄い量産ページを「強化／統合／noindex」に三分類（データが順位ゼロを示した群は対象）。クロール・評価を勝てる面に集中。
5. **被リンク/指名**: データレポートの引用獲得（既存の無料エンジン）＋PRで指名検索の種まき。

### P2（四半期）— 事業価値の最大化と売却
7. **リードジェンB2B化**（クリック量が支える領域で、解体/遺品整理の地場業者にリード販売＝査定アフィより高単価・意図一致）。

## 7. エグジット準備（DD対策＋資産パッケージ＋買い手像）

**DD対策（買い手の減点を消す）**:
- 数字の整合（GA4/GSC/A8）と"盛らない"開示。トラフィックの実在・質を証明。
- 法務（非弁/非税理士・景表法・PII非保存）の遵守ログ。資格表示の厳密性。
- 自動化の再現性（誰が動かしても回る）を文書化。

**売り物（アセット）の見せ方**:
- **1,726自治体の独自DB**（更新運用つき）＝他に無いデータの堀。
- **pSEO×自動化パイプライン**（記事/データ/インデックス/被リンク）＝再現可能な顧客獲得装置。
- **3,500ページのインデックス在庫＋ドメイン＋ブランド**。
- **（検証済みの）トラフィックと、意図一致で伸びる収益/リードのトレンド**。

**戦略的買い手の像**:
- 解体・空き家のマッチング/ポータル、遺品整理・不用品回収のチェーン/FC、不動産（買取/査定）、終活・葬祭・士業プラットフォーム、自治体向けGovTech、住生活系メディアのロールアップ。
- 彼らが欲しいのは"EBITDA"でなく**「自社サービスへ送客する獲得チャネル＋データ」**。だからこそ収益が小さくてもアセット価値で3,000万円の物語が成立しうる。

**読み解き**:

## 付録B：CTA（出口）の実態 — 意図出し分けは概ね実装済み

コード精査の結果、当初仮説（査定を全面に出して不一致）は**大部分が既に解消済み**：
- **空き家/補助金/area ページ** → `RealEstateAppraisalCard`＋StickyCTAが査定（✅ 空き家オーナーに意図一致）。
- **cleanup ページ（粗大ゴミ/不用品・noindex）** → `CleanupAffiliateCard`（不用品回収アフィリエイト：首都圏=アールクリーニング/全国=ECOクリーン）。検索流入はnoindexで無し。
- **情報記事** → Stickyは `/tool/optimizer` の診断（ソフトリード）。

**→ 修正版の結論**: 換金が伸びない主因は「CTAの配置ミス」ではなく、**(1) 流入の実在/質（GA4×GSC乖離）**と、**(2) 査定は"今すぐ売る高intentオーナー"向けだが、補助金クラスタは情報段階の超ローカル探索者**であること。
- 処方は**配置変更より"intent段階に合わせた出口"**：補助金/解体費用ページは**①無料PDF・メール（Zero-Party Data）で情報段階を捕捉→後追い、②"解体一括見積もり"の出口を併設**（"解体費用"intentに土地査定より一致）、そして**③権威でクラスタを1ページ目に上げて母数を作る**。
