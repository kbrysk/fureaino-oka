## 1. 診断（Googleアルゴリズム観点）
数千の「[市]×[コモディティ作業]」薄ページが pos40-95・0クリック＝**HCSの"scaled content"シグネチャ**。
サイト全体の品質シグナルを押し下げ、勝てるページまで56位に沈めている。**順位が上がらない根因は"薄ページ過多"**。

## 2. 4観点での結論（重要：noindex ≠ 削除）
- **noindexしてもページはサイトに残り、ユーザーは到達できる**（検索結果から外れるだけ）。
- **SEO**：0クリック薄ページを刈る→失う流入0／勝てるページに品質シグナル集中。理想は"都市ハブへ統合"。
- **Google Ads（商業価値）**：粗大ゴミ/遺品整理[市]流入は低CV（安い地元業者探し）。査定/補助金/固定資産税が高CV。**商業損失ほぼ0**。
- **ユーザー**：網羅情報は"1都市1ハブ"で満たせる（むしろ親切）。薄ページ50位→即離脱が今のUX。
- **起業家/EXIT**：枚数は虚栄指標。買い手DDは薄ページ過多を"次のコア更新で飛ぶリスク"＝減点。刈れば**防御可能な資産**。堀は「独自補助金データ＋実家じまいニッチ1位」。

## 3. コンセプトの再定義
> ❌ 旧：量産pSEOで全自治体×全作業を網羅（＝失敗中・56位）
> ✅ 新：**独自データに裏打ちされた「実家じまい × 空き家補助金」の権威メディア**。
> 　　　自治体データ＝武器／コモディティ作業（粗大ゴミ・遺品・解体費用）＝**ユーザー回遊用の補助コンテンツ（noindex・ハブ集約）**。

## 4. 実施済み（本ラン）
- **garbage（粗大ゴミ[市]）を noindex 化**（generateMetadata両分岐・cleanupと同型）＋area sitemapから除外。本番反映済。
- **統合の確認**：base都市ハブは既に「粗大ゴミ申込（各市の公式URL `area.bulkyWasteUrl`）・出し方ルール・解体費用シミュ・補助金・遺品整理」を内包し /garbage /cost /subsidy へリンク済み。
  ＝**garbageをnoindexしても情報・公式URLは indexed な base ハブに残る。ユーザー価値は不変。**

## 5. 段階ロードマップ（計測しながら）
- **P0-a（完了）**：cleanup(既)＋garbage を noindex。
- **P0-b（測定後に判断）**：cost の「解体費用[市]」ロングテール、base内のコモディティ節の扱い。**garbageの効果を3-4週測ってから**。
- **P1（並行・勝者投資）**：実家じまい[市]（base＝1位）・補助金[町村]・固定資産税[市](tax-sim) への内部リンク集中＋データ鮮度維持。
- **P2（稼働中）**：被リンク（送信55件＋データ資産）。中部新生不動産から初の前向き返信→愛知データ提供済。
- **P3（上位化後）**：CTR最適化。

## 6. 効果測定（数週間後に再取得して比較）
GSCを3-4週後に再取得し、本ベースライン（§0）と比較。**改善のサイン**＝
1. **平均掲載順位 56.4位 → 改善**（薄ページがindexから抜け、勝てるページが浮上）。
2. **garbage系クエリ（粗大ゴミ/遺品整理[市]）の表示がGSCから減少**（noindex浸透の証拠）。
3. **実家じまい[市]・補助金[町村]・固定資産税[市]クラスタの平均順位が上昇**。
4. **CTR 0.38% → 上昇**（無価値なPC6ページ目表示が減るため分母改善）。
- 取得：`npm run gsc:report`（※現在ADC未認証＝要再認証）。手動DLでも可（本doc§0が比較基準）。
- 注意：HCS/再評価は**数週間〜数か月**。短期の上下に一喜一憂しない。
