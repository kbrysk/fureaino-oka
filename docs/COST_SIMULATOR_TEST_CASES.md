# 解体・片付け費用シミュレーター テストケース報告

## 計算式（実装）

$$
TotalCost = \max(0,\ (Area \times UnitPrice \times StructureFactor \times HandMult) + DisposalCost - SubsidyAmount)
$$

- **Area**: 20 / 30 / 40 / 50坪
- **UnitPrice**: 地域単価（基準 2.8万円/坪 × 地域係数。`regional-stats.csv` の `landPrice` で補正、0.85〜1.25）
- **StructureFactor**: 木造 1.0 / 鉄骨 1.2 / RC 1.5
- **HandMult**: 手壊し（重機搬入困難）時 1.28、それ以外 1.0
- **DisposalCost**: 少 10万 / 中 25万 / 多 45万
- **SubsidyAmount**: `subsidyInfo.maxAmount` から抽出した最大万円（複数ある場合は最大値を採用）

---

## 札幌市（補助金あり）

- **補助金**: 「通常型は上限50万円、地域連携型は上限150万円」→ **150万円** を控除
- **地域**: 豪雪・積雪のため `hasSnowRegion=true` → 冬期割増の注釈を表示
- **regional-stats**: `hokkaido-sapporo` が CSV に無い場合は地域係数 1.0（基準単価 2.8万/坪）

### 例: 30坪・木造・荷物普通・重機可

| 項目           | 計算 |
|----------------|------|
| 解体ベース     | 30 × 28,000 × 1.0 = 84万円 |
| 片付け         | 25万円 |
| 小計           | 109万円 |
| 補助金控除     | －150万円（上限まで） |
| **実質負担額** | **0円**（補助金で賄う想定） |

※ 実際の表示では「補助金 150万円 を差し引いています」と注釈。実質負担が 0 以下なら 0 円表示。

### 例: 40坪・木造・荷物多い・重機可

- 解体: 40 × 28,000 = 112万、片付け 45万 → 157万
- 補助金 －150万 → **実質 約7万円**

札幌では補助金ありのため、同じ条件でも**実質負担額が大きく下がる**。

## 長崎市（特殊立地）

- **補助金**: 「—」→ **0円**（控除なし）
- **地域**: 坂・階段が多いため `hasNarrowAccess=true` → 「重機が入れない」を**デフォルトで推奨**し、該当注釈を表示
- **手壊し**: チェック時 1.28 倍（重機が入れない想定）

### 例: 30坪・木造・荷物普通・重機可

- 解体: 30 × 28,000 × 1.0 = 84万、片付け 25万 → **約109万円**
- 補助金なしのためそのまま 109万円。

### 例: 30坪・木造・荷物普通・**手壊し（重機不可）**

- 解体: 30 × 28,000 × 1.0 × **1.28** = 107.52万、片付け 25万 → **約133万円**
- 長崎のように路地・階段で重機が入れない場合は、**約24万円程度上乗せ**される想定。

長崎では **補助金なし ＋ 手壊し推奨** のため、同じ坪数・構造でも札幌（補助金あり）より**概算が高く出る**。

## 比較まとめ

| 条件           | 札幌市（補助金150万） | 長崎市（補助金なし・手壊し） |
|----------------|------------------------|------------------------------|
| 30坪・木造・中 | 実質 0円〜             | 約133万円                    |
| 40坪・木造・多 | 実質 約7万円            | 約205万円（手壊し時）       |

- **札幌**: 補助金で実質負担が小さく、冬期割増の注釈で地域特性を表示。
- **長崎**: 補助金なし・手壊しデフォルト推奨で、立地による費用増を反映。
