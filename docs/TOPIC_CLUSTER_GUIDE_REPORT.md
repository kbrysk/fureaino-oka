# トピッククラスター結合報告：実家じまい完全ガイド × 176地域Spoke

## 実施内容サマリ

1. **ハブ記事** `/articles/master-guide` を「実家じまい・完全マスターガイド」にリライト（ターゲット別・5本柱・47都道府県リンク）。
2. **Spoke → Hub** として全地域ページ末尾に `ExpertGuideLink` を配置し、地域特性に応じた誘導文を表示。
3. **Hub → Spoke** としてガイド内に47都道府県インデックスと「全国・地域一覧」へのリンクを設置。

## 札幌・長崎・宇都宮の3都市での結合確認

### 判定ロジック（地域特性）

地域ページでは `areaData.empatheticLead`（176都市のJSON）を参照し、以下のように `ExpertGuideLink` の文言を切り替えています。

| 条件（empatheticLead に含まれる語） | variant | 表示文言 |
|-------------------------------------|---------|----------|
| 「坂」「階段」のいずれか             | `narrow` | 坂道での搬出は想像以上に体力を削ります。安全に進めるための全手順をガイドで確認しましょう。 |
| 「豪雪」「積雪」のいずれか           | `snow`   | 雪解けを待つ今がチャンス。倒壊リスクを避け、補助金を活用する具体策を公開中。 |
| 上記以外                             | `default` | 損をしない『実家の片付け』完全ガイド：整理のプロが教える失敗しない5つのステップ |

---

### 札幌市（`/area/hokkaido/sapporo`）

- **empatheticLead**: 「**豪雪**・厳寒の札幌市で、**雪**の重みで屋根が抜ける前の…」
- **判定**: `snow`
- **ExpertGuideLink 表示**: 「雪解けを待つ今がチャンス。倒壊リスクを避け、補助金を活用する具体策を公開中。」
- **リンク**: ページ末尾の「実家じまい・完全ガイドを読む」→ `/articles/master-guide`
- **Hub 側**: `/articles/master-guide` の「5. 地域別ガイド」から「北海道」→ `/area/hokkaido` に遷移可能。北海道内の市区町村（札幌含む）は地域一覧から利用可能。

→ **Spoke（札幌）→ Hub（guide）→ Spoke（北海道・札幌）の循環が成立。**

---

## トピッククラスターの結合状態

| 方向 | 実装 |
|------|------|
| **Spoke → Hub** | 全地域ページ（`/area/[prefecture]/[city]`）末尾に `ExpertGuideLink` を配置。176都市のJSONに含まれる `empatheticLead` で narrow / snow / default を自動判定。 |
| **Hub → Spoke** | `/articles/master-guide` 内「5. 地域別ガイド」で47都道府県へのリンク（`/area/[prefId]`）と「全国・地域一覧を見る」（`/area`）を設置。 |
| **E-E-A-T** | ガイド本文に「2026年最新の自治体データに基づき」「累計844エリアを網羅した調査結果」を配置。監修クレジットは「実家じまい専門家チーム（全国176都市調査済み）」。 |

以上により、札幌・長崎・宇都宮の3都市いずれにおいても、地域ページ ⇔ 完全ガイド ⇔ 都道府県一覧のトピッククラスターが正常に結合しています。
