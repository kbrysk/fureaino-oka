# 内部リンク・トピッククラスター SEO監査レポート

**対象サイト**: 生前整理支援センター ふれあいの丘  
**監査観点**: 独立ページの有無、サイト全体の内部リンク最適化、Googleクローラーの巡回しやすさ、トピッククラスター構造  
**実施日**: 2026年2月

---

## 1. 抽象的・戦略的観点

### 1.1 トピッククラスターの設計

サイトは以下の3つのクラスターを軸に設計するのが適切です。

| クラスター | Hub（親玉） | Spoke（詳細） |
|-----------|-------------|----------------|
| **実家じまい** | `/guide`（実家じまいの進め方 全手順） | ツール: 実家じまい力診断 / 記事: 説得・業者選び / 事例 |
| **遺品整理・費用** | `/cost`（間取り別 遺品整理費用相場） | 詳細: `/cost/layout/[slug]` / 辞書: 仏壇・タンス等 `/dispose/[slug]` |
| **相続・手続き** | 記事・ツール群 | 法定相続分シミュレーター / 空き家税金シミュレーター / 相続準備力診断 |

**原則**: どのクラスターから入っても、最終的に「診断・相談」というゴール（蜘蛛の巣の中心）に誘導できるようにする。

### 1.2 クローラビリティの原則

- **オーファンページの排除**: すべての公開URLが、トップ・ナビ・フッター・本文・sitemap.xml のいずれかから**3クリック以内**で発見可能であること。
- **サイトマップの網羅**: 静的ルートに加え、動的ルート（間取り別費用・捨て方カテゴリ・品目・地域・記事）をすべて sitemap に含め、クロール予算を有効に使う。
- **アンカーテキストの具体化**: 「こちら」「詳細へ」は避け、「空き家税金シミュレーター」「間取り別 遺品整理費用相場」のようにリンク先の内容が分かる文言にする。

### 1.3 リンクジュースの配分

- **ハブページ**（`/`, `/guide`, `/cost`, `/dispose`, `/tools`）には、各クラスターの Spoke へのリンクをまとめて配置する。
- **Spoke ページ**（記事・品目・間取り別費用・地域）からは、必ずハブまたは診断・相談へ戻るリンクを設ける。
- フッターは「全重要ページを1クリックで発見可能」にする役割を維持する（今回も実施済み）。

## 2. 具体的な問題点と実施した改善

### 2.1 孤立ページ（オーファン）の解消

| ページ | 問題 | 実施した対策 |
|--------|------|--------------|
| `/region/[都道府県]/[市区町村]` | サイト内のどこからもリンクがなく、sitemap 経由のみ | **地域詳細**（`/area/[都道府県]/[市区町村]`）に、同一地域の region が存在する場合のみ「〇〇で安く遺品整理する裏技を読む」リンクを追加。area と region のデータが一致する市区町村で内部リンクが張られる。 |
| `/guidebook/jikka-jimai` | どこからもリンクされていなかった | **ガイドブックゲート**（`/guidebook`）に「メール登録なしでウェブ上で全編を読む」リンクを追加。登録済みユーザー向けに「実家じまい完全ガイドをウェブで読む」ボタンを追加。 |

### 2.2 サイトマップの拡充

- **追加した静的ルート**: `/cost`, `/dispose`, `/about`, `/senryu`, `/senryu/submit`, `/guidebook/jikka-jimai`、および全ツール個別URL（inheritance-share, hoji-calendar, digital-shame）。
- **追加した動的ルート**:
  - `/cost/layout/[slug]`（間取り別費用の各ページ）
  - `/dispose/category/[categorySlug]`（捨て方カテゴリ一覧）
  - `/dispose/[slug]`（捨て方品目ページ）

これにより、クローラーが全ページを確実に発見しやすくなり、インデックス効率が向上します。

### 2.3 トピッククラスターの強化（クロスリンクの追加）

| ページ | 追加したリンクブロック |
|--------|------------------------|
| **記事一覧**（`/articles`） | 「あわせて役立つコンテンツ」：間取り別費用相場・捨て方辞典・実家じまいの進め方・実家じまい力診断 |
| **記事詳細**（`/articles/[slug]`） | 「このあとできること」に 間取り別費用相場・捨て方辞典・実家じまい力診断 を追加 |
| **ツール一覧**（`/tools`） | 「あわせて読む」：実家じまいの進め方 全手順・間取り別 遺品整理費用相場・捨て方辞典 |
| **捨て方カテゴリ**（`/dispose/category/[slug]`） | 「部屋ごと・家全体を片付けたい方へ」：間取り別費用相場・実家じまい力診断 |
| **地域トップ**（`/area`） | 「実家じまい・遺品整理の費用と進め方」：間取り別費用相場・実家じまいの進め方・実家じまい力診断 |

※ `/guide` と `/cost` には既にクラスター間リンクが実装済み。捨て方品目ページには `DisposeToCostCrossLink` で費用・診断へのカード型リンクが設置済み。

### 2.4 その他の推奨（今後の検討）

- **記事本文中のアンカー**: 記事JSONの `body` 内に「こちら」「詳細へ」がある場合は、リンク先が分かる文言（例: 「空き家税金シミュレーターで計算する」）に変更する。
- **地域トップから region への一括入口**: `/area` と `/region` が別データのため、地域トップに「市区町村別・安く遺品整理する裏技記事」のようなテキストリンクで region の代表URLや一覧ページを設けると、region クラスター全体のクロールがさらに安定する。
- **指名検索・ブランド化**: 川柳のSNS展開や「フクロウのお悩み相談室」など、コンテンツ施策は別ドキュメントで整理することを推奨。

## 4. まとめ

- **独立していたページ**: `/region/...` は地域詳細からの条件付きリンクで、`/guidebook/jikka-jimai` はガイドブックゲートからのリンクで、いずれもサイト内から発見可能にした。
- **サイトマップ**: 主要静的ルートと、間取り別費用・捨て方カテゴリ・品目をすべて sitemap に含め、クローラビリティを強化した。
- **トピッククラスター**: 記事一覧・記事詳細・ツール一覧・捨て方カテゴリ・地域トップに、ハブ（guide / cost / dispose）および診断・相談へのクロスリンクを追加し、「蜘蛛の巣」型の内部リンク構成を整えた。

今後の更新では、新規記事・新規品目を追加する際に「関連リンク」で必ず cost / dispose / guide / 診断のいずれかへ誘導することをルール化すると、クラスター構造が維持されやすくなります。
