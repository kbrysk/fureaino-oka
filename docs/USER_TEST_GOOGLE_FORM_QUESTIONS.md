# クラウドワークス ユーザーテスト用 Googleフォーム 設問案

**対象サイト**: 生前整理支援センター ふれあいの丘  
**目的**: サイトを網羅的に利用いただいた上で、改善提案を収集する  
**報酬**: 50円程度（フォーム完了後のサンクスページに表示する「合言葉」をクラウドワークスで登録）

### 注意書き（文章の冒頭に記載）

- **「サイトを見ていないと判断した場合は、非承認とさせていただきます」** をフォーム説明文の冒頭に記載する。

### アンケート設計の方針

1. **すべての質問を必須回答にする**
2. **一つ一つのコンテンツ・ツールについて、利用後のアンケート回答を求める**  
   各ツール・コンテンツごとにセクションを設け、「まずそのコンテンツを利用 → 利用した感想を必ず記入」の流れにする。次のセクションでは別のツールの利用と感想を求める。
3. **各セクションの内容をじっくり読んだ上で回答してもらう**  
   各セクションの冒頭に「このコンテンツを実際にご利用のうえ、以下の質問に回答してください」と明記する。

---

## フォームの構成（自動作成スクリプトで生成される順序）

1. **説明・同意**（注意書き＋すべてのコンテンツを利用する旨の同意・必須）
2. **属性（年齢）**（必須）
3. **各ツール・コンテンツごとのセクション**（22件）  
   各セクション：「〇〇を利用したうえで回答してください」→「〇〇を利用した感想をお書きください」（必須）
4. **全体を通しての改善提案**（必須）
5. **合言葉確認**（サンクスページに表示）

## 1. 説明・同意

### 設問1-1（説明文・段落）

```
このアンケートは「生前整理支援センター ふれあいの丘」のサイト改善のため、
実際にサイトをご利用いただいた方の声を集めるものです。

【お願い】
・指定のサイトをできるだけ多くのページにアクセスし、無料ツールや記事なども
  実際に触ってから回答してください。
・回答完了後、表示される「合言葉」をクラウドワークス上の指定の場所に
  入力していただくことで報酬（50円程度）をお支払いします。
・所要時間の目安：サイトの閲覧 15〜30分 + アンケート回答 約5分

サイトURL: （ここに本番URLを記載）
```

### 設問1-2（チェックボックス・必須）

**質問**: 以下の内容に同意した上で、サイトを一通りご利用のうえ回答しますか？

- [ ] 指定URLのサイトにアクセスし、トップページ以外にも複数のページ・ツールを利用した
- [ ] 上記に同意し、回答を進める

※ 両方にチェックがないと次へ進めないようにする（または「同意する」1項目のみでも可）

## 2. 属性（年齢）

### 設問2-1（ラジオボタン・必須）

**質問**: あなたの年齢層を教えてください。

- 30代以下
- 40代
- 50代
- 60代
- 70代以上

※ 40〜60代をメインに募集する場合は、説明文で「40代〜60代の方を優先して募集しています」と記載しても可。

## 3. サイト利用確認（どこまで見たか）

### 設問3-1（チェックボックス・必須）

**質問**: 以下のうち、今回アクセスして内容を確認したページ・機能にすべてチェックを入れてください。（複数選択可・できるだけ多くお願いします）

**トップ・基本**
- [ ] トップページ（ホーム）
- [ ] ふれあいの丘について（会社・サービス紹介）

**無料ツール**
- [ ] 無料ツール一覧ページ
- [ ] 空き家税金シミュレーター
- [ ] 実家じまい力診断
- [ ] 空き家リスク診断
- [ ] 法定相続分シミュレーター
- [ ] 相続準備力診断
- [ ] デジタル遺品リスク診断
- [ ] 法要カレンダー
- [ ] 資産・査定の目安
- [ ] チェックリスト（生前整理の進捗）
- [ ] 地域別（粗大ゴミ・遺品整理の地域ページ）

**コンテンツ・情報**
- [ ] はじめかた・進め方（実家じまいの手順）
- [ ] 費用の目安（ cost 関連）
- [ ] 処分・捨て方（品目別など）
- [ ] 記事一覧または個別の記事（1本以上）
- [ ] ガイドブック（実家じまい等）

**その他**
- [ ] 専門家への無料相談（お問い合わせ・紹介）
- [ ] 資産・査定の見える化（持ち物登録）
- [ ] エンディングノート
- [ ] 上記以外のページ（フッターのリンク等）

※ 「少なくとも〇個以上にチェックを入れた方のみ報酬対象」とする場合は、フォームの「回答の検証」で最小選択数を設定できます。

### 設問3-2（短文・任意）

**質問**: 特に「役に立った」「わかりやすかった」と感じたページやツール名を1つ教えてください。（任意）

- 回答形式: 短文（1行）

## 4. 全体の印象・使いやすさ

### 設問4-1（ linear scale 1〜5 または 1〜7・必須）

**質問**: サイト全体の「使いやすさ」を1〜5で評価してください。（1＝とても使いにくい、5＝とても使いやすい）

- 1 / 2 / 3 / 4 / 5

### 設問4-2（ linear scale 1〜5・必須）

**質問**: サイトの「見た目・デザイン」の印象を1〜5で評価してください。（1＝あまり良くない、5＝とても良い）

- 1 / 2 / 3 / 4 / 5

### 設問4-3（ラジオボタン・必須）

**質問**: このサイトは「生前整理・実家じまい」の情報を探している人に、役に立つと思いますか？

- とても役に立つと思う
- ある程度役に立つと思う
- あまり役に立たないと思う
- 役に立たないと思う

### 設問4-4（チェックボックス・任意）

**質問**: 当てはまるものをすべて選んでください。（複数選択可）

- 欲しい情報がすぐに見つかった
- メニューやリンクの場所がわかりやすかった
- 文字の大きさや色が読みやすかった
- スマートフォンでも使いやすかった
- 無料ツールが多くて良かった
- キャラクター（フクロウ）が親しみやすかった
- 情報が多すぎて迷った
- どこをクリックすればよいかわかりにくかった
- その他（設問4-5で記述）

### 設問4-5（段落・任意）

**質問**: 設問4-4で「その他」を選んだ方、または追加で伝えたいことがあればお書きください。

- 回答形式: 段落（長文可）

## 5. 改善提案（自由記述）

### 設問5-1（段落・必須）

**質問**: サイトをより良くするための「改善提案」を自由にお書きください。例：わかりにくかった点、こうしたらよいと思う点、追加してほしい機能や情報、誤字・表示の不具合など。

- 回答形式: 段落（なるべく具体的に）
- 記入例: 「〇〇のページで、△△という説明があるとわかりやすい」「スマホで〇〇のボタンが押しづらかった」など

### 設問5-2（段落・任意）

**質問**: その他、運営者へ伝えたいことがあればお書きください。

- 回答形式: 段落

## 6. 合言葉の案内（サンクスページに表示する文面）

フォームの「回答の確認」または「送信後のメッセージ」に、以下のような文を表示してください。

```
ご回答ありがとうございました。

【報酬受け取りの手順】
1. この画面に表示されている「合言葉」をコピーするか、メモしてください。
2. クラウドワークスの該当タスクページに戻り、指定の入力欄に合言葉をそのまま貼り付けて提出してください。
3. 合言葉が一致した方に、報酬をお支払いします。

今回の合言葉はこちらです →

【 ふれあいテスト2025 】  
（※ カギ括弧も含めてそのまま入力してください。年度等は適宜変更可）

サイト改善にご協力いただき、ありがとうございました。
```

※ 合言葉は毎回変える場合は、フォームの「回答に応じてメッセージを変える」や、スプレッドシート連携で自動採番するなども検討できます。最初は固定の1つで運用するのが簡単です。

## フォーム設定のポイント

| 項目 | 推奨 |
|------|------|
| 必須項目 | 同意、年齢、利用確認（3-1）、使いやすさ（4-1）（4-2）（4-3）、改善提案（5-1） |
| 1ページあたりの設問数 | 5問前後で区切ると離脱が減りやすい |
| 回答時間の目安 | 冒頭に「約5分」と記載 |
| 合言葉の表示 | 「回答を送信」後のサンクスページに大きく表示 |
| 検証 | 設問3-1で「最低〇個選択」を設定すると、しっかり見た人だけ報酬対象にしやすい |

## クラウドワークス側のタスク説明に載せる文面（例）

```
【お願い】
1. 下記URLの「生前整理支援センター ふれあいの丘」にアクセスし、
   トップ・無料ツール・記事・はじめかた・費用など、できるだけ多くのページを
   実際にご覧ください（目安：15〜30分）。
2. サイトを一通りご利用のうえで、案内するGoogleフォームのアンケートに回答してください。
3. フォーム送信後に表示される「合言葉」を、本タスクの指定入力欄にそのまま入力して提出してください。
4. 合言葉が一致した方に報酬をお支払いします。

対象：40代〜60代の方を優先していますが、他の年齢の方も応募可能です。
報酬：50円（税込）
所要時間の目安：サイト閲覧 15〜30分 + アンケート 約5分
```

## 自動作成スクリプト（Google Apps Script）

手作業でフォームを作りたくない場合は、**スクリプトで一括作成**できます。

- **スクリプト**: `scripts/create-user-test-form.gs`
- **使い方**: `scripts/README.md` を参照

1. https://script.google.com/ で新規プロジェクトを作成
2. `create-user-test-form.gs` の内容を貼り付け
3. 先頭の `CONFIG` で `SITE_URL` と `AIKOTOBA` を編集
4. 関数 `createUserTestForm` を選択して「実行」
5. 初回のみ権限を許可し、実行ログに表示されるフォームURLを開く

実行すると、上記の設問がすべて入ったフォームが瞬時に作成されます。
