# 生前整理キーワード グルーピング・コンテンツ提案（SEO観点）

ラッコキーワードのサジェスト一覧と、各キーワードで検索した際の上位表示記事（約10記事）の傾向を踏まえた、コンテンツ用キーワードグルーピング案です。

---

## 前提：上位記事の傾向（共通）

- **検索意図**：多くのキーワードで「解説＋実務（やり方・リスト・費用）」の両方を満たす記事が上位。
- **構成**：見出しで「〇〇とは」「メリット・デメリット」「進め方・やり方」「年代別」「業者・費用」などを網羅している記事が強い。
- **文字量**：3,000〜6,000字程度の解説＋リスト／表が多く、薄い記事は上位に少ない。
- **CV導線**：遺品整理・不用品回収・買取・終活サービスへの導線を持ちつつ、本文は中立的な解説が多い。

## グルーピングしない（スキップしてよい）キーワード

- **地域名のみ**（例：生前整理 大阪、横浜、京都、岡山…）  
  → 地域別LPや地域ブログで別対応。全国記事には無理に詰めない。
- **ドラマ・有名人・商品名**（例：生前整理 ドラマ、〇〇 生前整理）  
  → 検索ボリュームが小さく、コンテンツの軸にしにくい。
- **資格・仕事系**（例：生前整理アドバイザー、資格、求人、開業）  
  → 資格取得者・業者志望向けで、一般ユーザー向けコンテンツの軸と意図が異なる。別サイト or 1コンテンツで軽く触れる程度。
- **表記ゆれ・誤変換**（例：生前整理 英语、断舍离）  
  → 対象外で問題なし。
- **極端にボリュームが小さいもの**（vol 0〜10で他とまとめにくい）  
  → 無理にグルーピングせず、メインKWの記事内で自然に触れれば十分。

## コンテンツ作成時の共通ポイント（上位表示を狙うため）

1. **タイトル・H1**  
   各グループのメインKWを前半に含める（例：「生前整理のやり方・進め方｜やることリストと…」）。
2. **見出し（H2/H3）**  
   サブKWをそのまま見出しに使う（例：「生前整理業者の選び方」「料金相場」）。
3. **本文**  
   対象KWを自然に含めつつ、上位記事と同程度の網羅性（項目数・具体例・表）を用意する。
4. **内部リンク**  
   グループ1→2→4→5、1→3、2→6→7 など、ファネルとテーマの近い記事同士を相互リンクする。
5. **地域KW**  
   全国記事では「地域名＋生前整理」を無理に詰め込まない。地域別ページで「〇〇（地域）の生前整理」を扱う。

## 記事別の骨子・構成案について

- **参照**：[docs/seo-article-outlines-seizenseiri.md](./seo-article-outlines-seizenseiri.md)
