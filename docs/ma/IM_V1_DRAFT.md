## 2. 譲渡対象資産（4つの柱）

- 全国47都道府県・**1,726市区町村**を網羅した構造化DB
- 解体・空き家補助金「制度あり」**844自治体**（上限額情報840件・自治体公式URL627件）
- 粗大ごみ・処分窓口データ **1,490自治体**
- 月次更新の調査運用手順書つき／Excel・CSV・JSON（API化済み構造）で即利用可能
- **再構築コスト試算: 自治体個別調査・構造化・検収で延べ1,000時間超＝外注換算500〜800万円＋構築期間6〜12ヶ月**

- B2Bライセンス販売の営業体制構築済み（商品資料・価格表・営業テンプレート一式を含む）

### 資産② 検索ポジション（実家じまい×空き家補助金ニッチ）
- 「実家じまい＋市区町村名」で**1位獲得実績**（大府市1位・清須市1.7位ほか）
- 空き家・解体補助金クラスタ: 平均35.5位・トップ20入り27クエリ（成長途上・伸びしろとして提示）
- **固定資産税シミュレーター: 平均掲載順位17.5位**（サイト最良資産・111URL）
- pSEO基盤: 市区町村×補助金/費用/税金の自動生成ページ群（2026年6月にHCS対応の品質清算実施済み＝薄ページnoindex・県別ハブ統合・「de-risk済み」）

- 死亡数: **160.5万人/年（2024年・過去最多）**＝遺品整理・実家じまい需要は構造的増加（厚労省）
