## 運用メモ
- **まず優先度A 10社**を B2B_SALES_EMAIL_TEMPLATES.md パターンAで送信（週5社・火曜運用）
- ★印4社（クラッソーネ/バリュークリエーション/じげん/鎌倉新書）は**M&A買い手候補と重複**——B2Bデータ契約が成立すれば、それ自体が売却交渉の価格根拠（継続収益＋関係構築）になる二重価値。ただしM&A接触戦略（FIRST_CONTACT_DRAFTS.md）と矛盾しないよう、クラッソーネ/バリュークリエーションは**M&A接触を優先**しデータ営業は後回し（同一相手に2つの提案を同時にぶつけない）
- 「要確認」「次回調査」の企業は、優先度A消化後にWeb調査で補完（次回セッション）
- 各社送信時は B2B_SALES_LOG.csv に必ず記録
