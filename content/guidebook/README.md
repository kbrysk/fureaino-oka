# ガイドブック（PDF）コンテンツ

## 2種類のガイド

| ガイド | 対象 | 主な内容 | 配布 |
|--------|------|----------|------|
| **生前整理 完全ガイドブック** | 本人（高齢者） | 優先順位チェック・財産一覧・エンディングノート・場所別の進め方・デジタル遺産・10の専門性 | メール登録でPDF送付（`/guidebook`） |
| **失敗しない実家じまい 完全ガイドブック** | 子・家族 | 期限一覧・最初にやること・兄弟で揉めない進め方・業者選び・空き家の選択肢・専門家マップ | LINE登録でPDF配布 |

## ファイル

- `data.ts` … 生前整理ガイド用データ（セクション・付録・専門家マップ等）
- `jikka-jimai.ts` … 実家じまいガイドのタイトル・サブタイトル定数
- `jikka-jimai-complete-guide.md` … 実家じまいガイド本文（PDF原稿）。このMarkdownをPDFに変換してLINE特典として配布する。

## PDF化

- 生前整理ガイド：`data.ts` を参照してサイト側または別ビルドでPDF生成
- 実家じまいガイド：`jikka-jimai-complete-guide.md` をPandoc・md-to-pdf・Vivliostyle等でPDF化するか、同内容をHTML/Reactでレンダーして印刷用CSSでPDF出力
