## 1. 加入する解体一括見積もりプログラム（おすすめ順）

| プログラム | 特徴 | 取り扱いASP（目安） |
|---|---|---|
| **クラッソーネ（解体・外構の無料一括見積もり）** | 解体特化・登録業者2,400社超・実績17万件超。意図一致度が最も高い | 公式提携／A8・afb・アクセストレード等で「クラッソーネ」を検索 |
| 解体無料見積ガイド／解体の窓口 系 | 解体一括見積 | A8/afb等で「解体 見積」検索 |
| ヌリカエ／リショップナビ系（解体対応） | リフォーム系だが解体も | A8等 |

### 加入手順
1. A8.net（既存アカウント）にログイン → プログラム検索で **「クラッソーネ」「解体 一括見積」** を検索。
2. 無ければ **afb（旧アフィリエイトB）／アクセストレード／もしもアフィリエイト**にも登録して同様に検索（クラッソーネは複数ASPに出ていることが多い）。
3. プログラムに**提携申請** → 承認後、「広告リンク」から**クリック計測URL（px...ejp?a8mat=...）**と、あれば**インプレッションgif（0.gif?a8mat=...）**を取得。

## 2. カードを有効化する（リンクを貼るだけ）

`app/components/DemolitionEstimateCard.tsx` の先頭2定数を差し替える：

```ts
// 加入後に取得した成果リンクを貼る
const DEMOLITION_AFFILIATE_CLICK = "https://px.a8.net/svt/ejp?a8mat=XXXXXXXX"; // ← クリックURL
const DEMOLITION_AFFILIATE_IMP   = "https://www18.a8.net/0.gif?a8mat=XXXXXXXX"; // ← インプレgif（任意）
```

- 設定すると、全補助金ページ（数千ページ）の解体見積カードが**一括で本物の一括見積もりCTAに切り替わります**。
- 未設定なら内部リンク（/akiya）にフォールバック（安全）。
- リンクには `rel="nofollow noopener"`（CtaButtonが外部URLに自動付与）＝アフィリエイトの正しい表記。

> リンクを共有いただければ、私（Claude Code）が差し替え＆ビルド＆デプロイまで実行します。
