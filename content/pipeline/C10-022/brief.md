## 0. 調査から確定した戦略（なぜこの設計か）

### SERP上位の table stakes（必須見出し）

「生前整理とは」上位12記事（バイセル・ALSOK・チェスター税理士法人・小さなお葬式・みんなの遺品整理等）に共通する必須論点。

1. **定義・意味**（1〜2文の明快な定義）：自分が元気なうちに身の回りの「物」「財産」「情報」を整理する行為。
2. **遺品整理・終活・老前整理との違い**：実施者・時期・目的の3軸で比較表がほぼ必須。
3. **メリット（3〜7項目）**：遺族の負担軽減・相続トラブル防止・快適な生活空間・人生の見直し・心の整理。
4. **デメリット（2〜5項目）**：時間・体力・費用がかかる、思い出の品を処分してしまうリスク。
5. **やること概要（4〜6項目の箇条書き）**：物の仕分け・書類財産の保管場所一覧・デジタル棚卸し・エンディングノート。
6. **いつから始めるか**：「思い立ったときが始め時」+体力・判断力がある40〜50代が多い傾向。
7. **始め方の最初の一歩**：今日できる具体的な1つの行動で締める。

これらは table stakes として外せない。

### 差別化の余地（上位が手薄な論点）

上位記事の多くが「死後のための整理」という後ろ向きフレームに依存しており、以下の論点が弱い。

1. **「今の暮らしを豊かにする」という前向きな再定義**：上位は「遺族のため」文脈が優勢で、「自分自身が今をスッキリ生きるため」という視点が薄い。生前整理アドバイザー視点の「心の整理も含む」という側面（デス・クリーニング思想との接続）を差別化ポイントとして空ける。
2. **「心の整理」という第5の柱**：物・財産・デジタル・エンディングノートという4分類だけでなく、「人生を振り返り何が大切かを見つめ直す」という内面的プロセスを1つの柱として明示している記事はほぼない。スウェーデン発祥の「デス・クリーニング（他者への思いやりを軸に整理する）」思想との接続は独自性が高い。
3. **「3行でわかる違いの先」の感情**：比較表は上位に存在するが、「なぜ違いを知ることが大切か」という感情的文脈（遺族に見せた比較表が実際の話し合いのきっかけになった等）まで書いた記事は少ない。
4. **業者トラブル・訪問買取注意喚起**：国民生活センター（2024年9月・訪問購入の深刻事例）と接続した消費者保護の文脈が上位記事にほぼない。生前整理を機に訪問購入トラブルに巻き込まれるリスクの注意喚起は独自切り口として有効。
5. **CV導線の差別化**：上位記事は外部送客なし（業者見積もりのみ）。チェックリスト・無料ガイドPDFというゼロパーティデータ獲得の導線を内部に設計できる。

### 2. H2：生前整理とは？意味・目的・定義

- **狙い**：検索意図(1)を直接回収。定義アンカー記事として最重要なセクション。明快な1〜2文の定義を冒頭に置き、「なぜするのか（目的）」を3点に整理する。断定調ではなく「〜とされています」「〜と言われています」の調整を徹底。
- **骨子**：
  - H3：生前整理の意味（定義）
    - 「自分が生きているうちに、自分の意思で身の回りの物・財産・情報・気持ちを整理する取り組み」という定義を1文で明示。
    - 法律・行政上の明確な定義はなく、一般的に「終活の一環」として位置づけられることが多い旨を補足。
  - H3：生前整理をする3つの目的
    - 目的1：遺族の負担を軽減する（時間・お金・判断の手間を減らす）
    - 目的2：自分自身が今をスッキリ暮らす（使わないものが減ると生活空間が快適になる）
    - 目的3：人生と向き合い「今大切なもの」を見つめ直す（心の整理）
    - 断定ではなく「〜とされています」「〜という方が多いようです」調。
- **公的出典候補**：特になし（定義・目的の概要のため一般論で構成。次のH2以降で公的出典を引用）

### 3. H2：生前整理・遺品整理・終活・老前整理の違い

- **狙い**：検索意図(3)を集中回収。副次KW「生前整理 終活 違い」「遺品整理 生前整理 違い」を確実に回収する。比較表（必須 table stakes）を1枚設置し、表の後に感情的文脈（「比較表を見ながら話し合いの糸口に」）を1〜2文添える。
- **骨子**：
  - H3：生前整理と遺品整理の違い
    - 比較表（4項目）：実施者・実施時期・目的・主な内容
      - 生前整理：本人自身 / 生前（元気なうち）/ 家族の負担軽減・自分らしい暮らし / 物の仕分け・財産の一覧化・デジタル整理等
      - 遺品整理：遺族・専門業者 / 死後 / 遺品・遺産の整理 / 不用品の処分・形見分け・買取等
    - 表の後に2文補足：「生前整理を済ませておくと、のちの遺品整理の量が格段に少なくなります。遺族が一人で判断しなければならない場面を減らすことが、最大のメリットのひとつです。」
  - H3：終活と生前整理の関係
    - 終活は「人生の終わりに向けた幅広い準備（葬儀・お墓・遺言・医療希望など）」の総称。生前整理はその中の「物・財産・情報の整理」に特化した行動。
    - 「終活の一部として生前整理がある」という包含関係を図または箇条書きで示す。
    - 終活全般のいつから・やること詳細は自社記事（C5-029）へ内部リンク。
  - H3：老前整理との違い（短め）
    - 老前整理は「老いる前に身の回りを整えて老後の暮らしをラクにする」ことが主目的。自分のQOLが主眼。生前整理は「死後に遺族が困らないよう」という他者への思いやりも含む点が異なる。
- **公的出典候補**：特になし（定義の整理のため一般論。ただし独自切り口スロットで差別化する）

### 4. H2：生前整理のメリット・デメリット

- **狙い**：検索意図(2)を回収。副次KW「生前整理 メリット」「生前整理 デメリット」を同一H2内で回収し、始めるかどうかを迷っている読者の背中を押す。デメリットを正直に示すことで信頼感を高める（E-E-A-T Trustworthiness）。デメリットの後に「対処法」1行を添えて前向きに着地させる。
- **骨子**：
  - メリット（4〜5項目の箇条書き）
    - 遺族の時間・精神的負担を大きく減らせる
    - 相続手続きがスムーズになる（財産の所在を把握しておくだけで大きな助け。具体的な手続きは専門家へ）
    - 使わないものが減り、今の生活空間が快適になる
    - 大切なものが明確になり、人生の優先順位が整う（心の整理）
    - 「家族に迷惑をかけたくない」という不安が和らぐ
  - デメリット（3〜4項目の箇条書き）
    - 時間と体力が必要。一気にやろうとすると挫折しやすい → 対処：「今日はここだけ」と範囲を小さく絞る
    - 思い出の品を処分してしまうリスク → 対処：迷ったものは「保留ボックス」に入れて半年後に判断
    - 業者に依頼する場合は費用がかかる → 対処：自分でできる範囲から始め、必要な部分だけ依頼
    - 悪質な訪問買取・押し買いに注意が必要（国民生活センター 2024年9月警告）→ 消費者ホットライン188
  - **公的出典候補**：国民生活センター「きっかけは訪問購入？犯罪まがいの深刻なトラブルにご注意を！」（2024年9月18日発表）（https://www.kokusen.go.jp/news/data/n-20240918_1.html）

### 5. H2：生前整理でやること——4つの領域

- **狙い**：検索意図(4)を回収。やること概要の箇条書きを提供しつつ、深掘りは各専門記事（master-guide・C10-040）へ送る。カニバリ回避のため「概要にとどめ・詳細は内部リンクで補完」の方針を厳守。財産・税務は「一覧化・保管場所の把握まで」で踏み込まない。
- **骨子**：
  - H3：（1）物の仕分け
    - 衣類・書籍・家電・思い出の品などを「使う／使わない／保留」の3分類で整理。一気にやらず場所を1か所ずつ進める。
  - H3：（2）書類・財産の保管場所を一覧化する
    - 通帳・保険証券・年金関係書類・不動産権利証などの「所在をメモしておく」だけで家族の負担が大きく変わる。具体的な税額計算や相続手続きは税理士・司法書士などの専門家へ。
  - H3：（3）デジタルの棚卸し
    - ネット銀行・証券口座・サブスクリプション・SNSアカウントなどのサービスリストを作る。国民生活センター（2024年11月「デジタル終活」）によると、パスワード不明で遺族がサブスク解約できないトラブルが増加している（https://www.kokusen.go.jp/news/data/n-20241120_1.html）。
    - PII保護注意：パスワードそのものを書き残さない（専用管理アプリを活用し保管場所のみ共有）。口座番号・暗証番号・PINを書面に残さない旨を伝える。
  - H3：（4）エンディングノートを書き始める
    - エンディングノートは遺言書と異なり法的拘束力はなく、医療希望・家族へのメッセージ・連絡先などを自由に書けるもの。「まず1ページだけ」から始めるハードルの低さを伝える。書き方の詳細は内部リンク（/guidebook）へ。遺言書の具体的文案・法的効力は弁護士へ（非弁遵守）。
  - 【フクロウ相槌スロット①】：「4つあると聞くと多く感じるかもしれないけど、まずは引き出し1つだけからでいいんだよ。」
- **公的出典候補**：国民生活センター「今から考えておきたい『デジタル終活』」（2024年11月20日発表）（https://www.kokusen.go.jp/news/data/n-20241120_1.html）

### 6. H2：生前整理はいつから始める？

- **狙い**：検索意図(5)を回収。「決まった年齢はない」を軸にしつつ「元気なうちが最良」の根拠を内閣府データで補完。上位の弱点は「何歳ベスト」の根拠が感覚論にとどまる点。公的データ（健康寿命ギャップ）を使ったロジックで差別化。C10-041（50代特化）との棲み分けは「概要のみ・詳細はC10-041へ送る」。
- **骨子**：
  - 始める年齢に決まりはない。「思い立ったときが始め時」が基本。
  - ただし、体力・判断力・時間の余裕が揃っているうちに始める方が余裕を持って進められる。
  - 内閣府「令和7年版高齢社会白書」によると健康寿命は男性72.57歳・女性75.45歳（令和4年）。一方、平均寿命との差は男性で約8年、女性で約12年（出典：内閣府 https://www8.cao.go.jp/kourei/whitepaper/w-2025/html/zenbun/s1_2_2.html）。「元気に動ける時間」を意識して今から少しずつ始める視点。
  - 年代別のきっかけ例（2〜3行の箇条書き）：退職・子の独立・親の介護経験など。
  - 50代・60代の生前整理の進め方詳細は `C10-041`（/articles/seizenseiri-50dai）へ内部リンク。
  - 【フクロウ相槌スロット②】：「縁起が悪いとは思わなくていいよ。今の暮らしをスッキリ整えるための行動なんだから。」
- **公的出典候補**：内閣府「令和7年版高齢社会白書（健康・福祉）」（https://www8.cao.go.jp/kourei/whitepaper/w-2025/html/zenbun/s1_2_2.html）
