## 0. 調査から確定した戦略（なぜこの設計か）

### 競合の壁と弱点（SERP分析）

上位記事（みんなの遺品整理・ブルークリーン・eco-life-smile・meetsmore・お助けうさぎ等）は以下を table stakes として固めている。

- 自分でできるかどうかの判断基準（水回りが使えるか・生活動線が残っているか・悪臭・害虫の有無）

- 心理・病気の背景（ためこみ症・認知症）への言及

### 差別化の余地（上位が手薄な論点）

1. **実家（親のゴミ屋敷）への家族の関わり方**：検索者の多くは「自宅ではなく親・実家のゴミ屋敷」で困っている層を含む。しかし上位記事の大半は「自分がゴミ屋敷を片付ける」視点で設計されており、「親に本人が動かない場合」「遠方の子が段取りを取る場合」「家族が費用をどう負担するか」の合意プロセスを丁寧に書いた記事は皆無に近い。
2. **ためこみ症・認知機能低下への責めないアプローチ**：心理・病気の背景に触れる記事は増えているが、「本人を責めず、どう接するか」「地域包括支援センターへの具体的なつなぎ方」まで掘り下げた記事は少ない。P4-039（物を捨てない親の心理）で培った文体・構成の強みを横展開できる差別化スロット。
3. **費用負担の家族間合意**：「誰が払うか」「遠方のきょうだいとどう分担するか」という論点は需要が高いが（SERP上の「実家 片付け費用 誰が払う」関連記事の存在が証拠）、ゴミ屋敷特化記事では手薄。
4. **CV導線の欠如**：競合は業者への直接送客のみ。自社地域別情報（/area）・チェックリスト・実家じまいガイドへの内部リンクを持つ記事は存在しない。

### 本記事のポジション

共感（本人も家族も責めない）→ 自分でできる範囲の確認 → 自分で片付ける手順 → 自分でvs業者の判断 → 業者に頼む費用の目安（間取り別）→ 業者選びの注意と許可業者確認 → 背景にある心理・ためこみ症と家族の関わり方 → 再発防止 → まとめ＋CVの流れで、**「ゴミ屋敷をどうにかしたいが量・心理・費用で困っている読者」の不安を解消しながら地域別不用品回収情報・実家じまいガイドへ橋渡しする**記事として設計する。

### 5. H2：業者選びで後悔しないための注意点

**狙い**：検索意図(4)を解決。無料回収・無許可巡回業者のトラブルを公的出典で裏付け、読者を守る情報として提供。E-E-A-T強化。

- H3：必ず確認｜一般廃棄物処理業の許可
  - 家庭ごみの回収には、市区町村の「一般廃棄物処理業許可」が必要（産業廃棄物許可のみでは不可）
  - 市区町村のHP・窓口で許可業者を確認できる
  - **公的出典**：環境省「廃棄物の処分に『無許可』の回収業者を利用しないでください」（https://www.env.go.jp/recycle/kaden/tv-recycle/qa.html）
- H3：「無料回収」「トラック積み放題」に潜むリスク
  - 国民生活センターへの相談件数は2021年度に2,000件超
  - 基本料金に加えて人件費・廃棄費等の名目で追加請求されるトラブルが多い
  - 訪問してその場で契約した場合でも8日以内なら特定商取引法によりクーリング・オフ可能
  - 不安を感じたら消費者ホットライン「188（いやや）」へ
  - **公的出典**：国民生活センター「不用品回収サービスのトラブル」（https://www.kokusen.go.jp/news/data/n-20221102_1.html）
- H3：見積もり時に確認すべき4つのポイント
  - 追加料金の発生条件が明記されているか
  - 処分先（最終処分場・リサイクル施設）の説明があるか
  - 作業後の写真報告があるか
  - キャンセル・クーリング・オフの対応を確認
