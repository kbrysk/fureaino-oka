## 0. 調査から確定した戦略（なぜこの設計か）

### 競合の壁と弱点（SERP分析）

上位記事（大野屋・小さなお葬式・イオンのお葬式・みんなの仏壇処分・長谷川仏壇等）は以下を table stakes として固めている。

- 閉眼供養（魂抜き・お性根抜き）の説明と、宗派による呼び方の違い（浄土真宗=遷仏法要）
- 処分方法の選択肢羅列（菩提寺・仏具店・専門業者・自治体粗大ごみ・遺品整理業者・買取）
- 費用の目安（閉眼供養のお布施3〜10万円程度・業者回収数千円〜数万円）
- 位牌・本尊の扱い（仏壇本体とは別に供養が必要）

### 差別化の余地（上位が手薄な論点）

1. **実家じまい文脈での仏壇問題**：「誰が引き取るか」「継ぐ人がいない」場合の親族・家族間の合意プロセス。「仏壇 処分」で来る読者の多くは実家じまいの中で困っているが、感情的合意の進め方を丁寧に書いた記事は皆無に近い。
2. **悪質業者・無許可回収への注意喚起**：無許可の不用品回収業者や訪問買取トラブルへの言及が上位記事では薄い。環境省の無許可業者注意喚起・国民生活センターの訪問購入トラブル情報を自然に引用できる。
3. **空き家に放置された仏壇の問題**：相続後に誰も住んでいない実家に仏壇が残っているケースへの具体的な対処手順（遠方からの段取り）が不足。
4. **CV導線の欠如**：競合は菩提寺・業者への直接送客のみ。自社チェックリストや地域別粗大ごみ情報への内部リンクを持つ記事は存在しない。

### 本記事のポジション

共感（罪悪感・不安に寄り添う）→ 処分前にすること（閉眼供養の一般的な流れ）→ 処分方法の選び方 → 費用の目安 → 位牌・本尊の扱い → 家族合意と注意点 → まとめ＋CVの流れで、**「仏壇をどうすればいいか分からない」読者の不安を解消しながら遺品整理・地域別粗大ごみ情報へ橋渡しする**記事として設計する。

### 2. H2：仏壇を処分する前に「閉眼供養」を行うのが一般的です

- **狙い**：検索意図(1)の核心。「魂抜き」「お性根抜き」という副次ワードをここで回収。読者の精神的不安を取り除いてから処分方法へ誘導する。
- **公的出典候補**：菩提寺・各寺院の案内という性質のため自治体等の公的出典はなし。「一般的に〜されている方が多い」「宗派や寺院によって呼び方・作法が異なるため、菩提寺や各寺院にご確認を」の留保を必ず入れる。
- **骨子**：
  - H3：閉眼供養（魂抜き・お性根抜き）とは
    - 仏壇を処分する前に、僧侶に読経していただき「仏様の魂を抜く（戻していただく）」儀式。処分後の仏壇はただの箱となり、ごみとして出すことへの精神的なハードルが下がる。
    - 呼び方は「閉眼供養・魂抜き・お性根抜き・閉眼法要」など宗派や地域で異なる。浄土真宗では「遷仏法要（遷座法要）」という名称で行われる場合が多い（仏壇に魂が宿るという考え方がない宗派のため、呼び方・意味合いが異なる）。
    - **宗教断定なし**：「〜という考え方が一般的です」「詳細は菩提寺や各寺院にご確認ください」のトーンを徹底。特定宗派・特定寺院を推奨しない。
  - H3：閉眼供養の一般的な流れ（ステップ形式）
    - ①菩提寺に相談する / ②日程・場所を決める（自宅か寺院か）/ ③当日、僧侶に読経していただく / ④お布施を渡す / ⑤仏壇の処分方法を選ぶ
    - 菩提寺がない場合：仏具店・遺品整理業者が僧侶を手配してくれるサービスもある旨を案内。
  - H3：閉眼供養のお布施の目安
    - 目安として3万〜10万円程度とされることが多いが、地域・寺院・付き合いの深さによって大きく異なる。事前に菩提寺に「どのくらいお包みすればよいでしょうか」と確認するのが最もスムーズ。
    - **断定しない**：「〇円が正解」という書き方をせず「目安として〇〜〇円程度といわれることが多いですが」＋留保を徹底。御車代・御膳料（各5,000〜1万円程度）が別途必要な場合もある旨も一言触れる。
- 【フクロウ相槌スロット】：「悩んでいる間も、ご先祖への想いは伝わっているよ。まず菩提寺に相談してみるだけでいいホー。」

### 3. H2：仏壇の処分方法を選ぶ — 5つの選択肢と向き不向き

- **狙い**：検索意図(2)の核心。処分方法を網羅しながら、各方法の「向いている状況」を一言で示すことで選択の手助けをする。table stakes を確実に押さえつつ、悪質業者への注意喚起で差別化する。
- **公的出典候補**：
  - 環境省「廃棄物の処分に無許可の回収業者を利用しないでください」(https://www.env.go.jp/recycle/kaden/tv-recycle/qa.html)
  - 国民生活センター「訪問購入のトラブル防止」(https://www.kokusen.go.jp/t_box/data/t_box-faq_qa2017_26.html)
  - 各市区町村の粗大ごみ・ごみ分別案内（.lg.jp ドメイン）
- **骨子**：
  - H3：①菩提寺・お寺に依頼する（最もスムーズな方法）
    - 閉眼供養からお焚き上げ・引き取りまで一連でお願いできる場合がある。まず菩提寺に相談するのが精神的にも安心。
    - 向いている人：菩提寺との付き合いがある・宗教的なけじめを重視している方。
  - H3：②仏具店・仏壇販売店に依頼する
    - 閉眼供養の僧侶手配から引き取り・処分を一括対応してくれる店舗がある。買い替えの場合は引取サービスが無料になるケースも。
    - 向いている人：菩提寺がない・仏具店との付き合いがある方。
  - H3：③専門の仏壇回収・処分業者を利用する
    - 「仏壇処分」専門業者が全国的に増えており、出張見積もり・閉眼供養の僧侶手配・引き取りをパッケージで提供。費用目安は数千円〜数万円（サイズ・地域による）。
    - 向いている人：菩提寺がない・費用を明確にしたい方。
    - **注意**：無許可の不用品回収業者は廃棄物処理法の観点から問題がある。「無料回収」をうたうチラシや軽トラ巡回業者のすべてが許可を持っているわけではない。一般廃棄物処理業の許可の有無を確認することが大切（出典：環境省）。
  - H3：④自治体の粗大ごみとして出す
    - 閉眼供養を済ませた後であれば、多くの自治体では仏壇を粗大ごみとして受け付けている。ただし仏壇を粗大ごみとして回収しない自治体もあるため、事前にお住まいの市区町村の公式サイト（ごみ・リサイクルのページ）で確認が必要。
    - 費用はごみ処理券代のみで済む（目安は数百〜2,000円程度・自治体によって異なる）。
    - 向いている人：費用を抑えたい・運び出せるサイズの仏壇の方。
  - H3：⑤遺品整理業者に一括依頼する
    - 実家の遺品整理と合わせて仏壇も一括で対応してもらえる。実家じまいの文脈では最も段取りがシンプル。
    - 向いている人：実家丸ごとの片付けが必要な方・遠方で手続きが難しい方。
    - **訪問買取への注意喚起**：遺品整理や仏壇処分を口実にした悪質業者が、貴金属・骨董品を強引に買い取るトラブルが報告されている。困ったときは消費者ホットライン「188」へ（出典：国民生活センター）。

### 4. H2：位牌・本尊・仏具の扱い方

- **狙い**：検索意図(4)。仏壇本体とは別に「位牌・本尊・仏具はどうするの？」という疑問は上位記事でも扱いが浅い。ここを丁寧に補足することで記事の信頼感を高める。
- **公的出典候補**：特定の公的機関の出典はないが「菩提寺にご相談ください」「各宗派の作法に従って」の留保を徹底。
- **骨子**：
  - H3：位牌の扱い
    - 仏壇の閉眼供養とは別に、位牌にも個別に閉眼供養（魂抜き）が必要と言われることが多い。まとめて菩提寺にお願いするのが一般的。不要になった位牌は寺院に永代供養を依頼するか、お焚き上げをしてもらう。
    - 浄土真宗など一部宗派では位牌の概念が異なる場合があるため、不明な点は菩提寺に確認を。
  - H3：本尊（ご本尊）の扱い
    - 本尊は仏壇の中で最も重要な仏様とされている。一般的には菩提寺に返還するか、お焚き上げをお願いする。仏壇業者によってはまとめて対応してくれる場合も。
  - H3：仏具・骨董品の価値確認は慎重に
    - 古い仏壇や仏具の中には骨董的な価値があるものも。一方で「仏具・骨董を買い取る」という訪問業者によるトラブルが報告されている。訪問買取を求められた場合は即決せず、まず消費者ホットライン「188」に相談することを検討してほしい。

### 5. H2：実家の仏壇問題は家族・親族と早めに話し合う（差別化の独自切り口 — expert-angle スロット）

- **狙い**：競合が完全に手薄な「実家じまい文脈での家族合意プロセス」を深掘り。仏壇を誰が受け取るか・供養をどうするかをめぐる家族間のすれ違いに寄り添う。ふれあいの丘のコアターゲット（40〜60代・実家じまい世代）に直接刺さる。
- **公的出典候補**：なし（家族コミュニケーション・心理の一般論）。内部リンクで実家じまいガイドへ接続。
- **骨子**：
  - H3：「誰が引き取るか」でもめないために
    - 仏壇は法律上「祭祀財産」に分類され、相続財産とは区別される。誰が引き継ぐかは法的に決まっているわけではなく、家族・親族間の話し合いで決める。処分を決める前に、まず関係者全員に意向を確認することが後のトラブル防止につながる。
    - **法務NG回避**：「祭祀財産の法律上の詳細については、弁護士・司法書士にご確認ください」の留保を入れる。相続税・分割協議には踏み込まない。
  - H3：遠方にいる・継ぐ人がいない場合の対処
    - 遠方在住で実家に足を運べない場合は、仏具店や遺品整理業者に状況を伝えた上でオンライン打ち合わせ＋出張対応を依頼する方法がある。継ぐ人がいない場合でも、永代供養という選択肢がある（菩提寺・霊園が寺院側で継続して供養を行う形式）。
    - 空き家に放置したままにしておくと管理費・雨漏り等のリスクもあるため、空き家の状況が心配な方は地域別の情報も参考に。
  - **【独自切り口スロット】**：expert-angle（大久保亮佑 生前整理アドバイザー視点）が後で埋める。現場で見た「仏壇を巡る家族の話し合いのきっかけになったエピソード」などを1つ入れるとE-E-A-T強化。

## 法務・倫理チェック

- [x] 非弁・非税理士：相続・税務に踏み込まない。祭祀財産の法的詳細は「弁護士・司法書士へ」の誘導にとどめる。
