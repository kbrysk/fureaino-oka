### Tier C：やらない（時間の無駄）

## 2. 実行手順（Claude Code・人手最小）

1. **発見（無料）**: `npm run links:mentions -- --dry-run`（要検索APIキー）か、**Claude CodeのWebSearchで上記クエリを回し**、`content/pipeline/outreach.csv` に候補追記。
2. **選別**: `link-director` で Tier A/B のみ優先度Aに。競合は `nofly`。
3. **下書き**: `outreach-writer` で1件ずつパーソナライズ（相手の既存記事に具体的に触れる）。
4. **自治体**: `npm run links:municipalities -- --kind public-sector --limit 20 --dry-run` → 下書き確認 → 送信は人間。
5. **計測**: `npm run links:profile`（GSC/Ahrefs CSVをinboxへ）で参照ドメイン純増を月次追跡。

## 4. 解体一括見積もり 出口（"解体費用"intentの換金・要A8加入）

データ上「解体費用 ◯◯市」クラスタは表示1,183（ただし順位53＝要権威）。土地査定より**解体一括見積**の方が意図一致。**現状A8に解体見積プログラム未連携**のため、以下を加入検討：
- **クラッソーネ（解体・外構の一括見積）** — 解体特化で意図一致が高い。
- **解体無料見積ガイド／解体の窓口 系** — 一括見積。
- **ヌリカエ／リショップナビ系（解体も対応）** — 既に競合だが提携可なら。

加入後、`RealEstateAppraisalCard` の隣に「解体一括見積カード」を追加し、補助金/解体ページで**査定（売る人）＋解体見積（解体する人）**を出し分け。私（Claude Code）が実装します。
