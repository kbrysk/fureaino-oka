## 0. 調査から確定した戦略（なぜこの設計か）

### SERP上位の table stakes（必須見出し）

検索上位（再良市場・プロアシスト東日本・神戸買取・ベストサーブ・東京相続相談ナビ等）の共通構成を分析すると、以下が必須論点として定着している。

1. **「50代はまだ早くない」の根拠提示**：体力・判断力・時間の三拍子が揃うのが50代という位置付け。
2. **メリット列挙**（家族の負担軽減・相続トラブル回避・心理的スッキリ・資産の見直し）。
3. **やること3〜5分類**（物の整理・書類・財産の把握・デジタル・エンディングノート）。
4. **進め方・続けるコツ**（一気にやらない・マイクロタスク化・家族と共有）。

これらは table stakes として外せない。

### 差別化の余地（上位が手薄な論点）

1. **「親の備えと自分の備えが重なる世代」という二重の文脈**：50代は親の介護・終活をサポートしながら、同時に自分自身の生前整理も意識し始める「二重の当事者」世代。上位記事のほぼすべてが「自分の生前整理」のみを論じており、親の終活との連動視点が欠落している。50代に固有の「サンドイッチ世代」の緊張感を描くことで独自の共感ポイントを作れる。
2. **「早いから少しずつできる」という逆転のフレーミング**：上位は「早い方がいい理由」を体力論で正当化するが、「時間があるからこそ1日1引き出しペースで10年かけられる」という余裕の論理に転換した記事はない。焦り→安心の感情移動が差別化になる。
3. **デジタル資産の具体性**：国民生活センター（2024年11月）が「デジタル終活」の必要性を発表。サブスク・キャッシュレス・ネット銀行などを「リスト化から始める」という具体手順が上位記事では薄い。
4. **CV導線**：競合は記事完結型（外部へ誘導しない）が大半。チェックリスト・エンディングノート・相続シミュレーターといった自社ツールへの内部リンクを設計することで滞在時間・コンバージョンで優位に立てる。

共感（50代で考え始めるのはちょうどいいタイミング）→ なぜ50代から（体力・時間・二重の当事者性）→ メリット → やること → 続けるコツ → CV（チェックリスト/エンディングノート/無料ガイドPDF）の黄金フォーミュラに沿い、**C10クラスタの中で「50代」「何歳から」「タイミング」の検索を全てカバーしながら、/checklsit・/guidebook・/tools/souzoku-prep への送客を担うサテライト記事**として設計する。

### 2. H2：「まだ早い」は間違い——50代から生前整理を始める理由
- **狙い**：検索意図(1)(2)を回収。「50代はまだ早い」という誤解を優しく崩し、「ちょうどいいタイミング」として背中を押す。上位の必須論点を確実に押さえる。
- **骨子**：
  - H3：体力と判断力が両立している今が動き時
    - 重い荷物の運搬・長時間の仕分け作業には体力が必要。認知症リスクが上がる70代以降では「もの」への判断が難しくなることも。今のうちに余裕を持って進められることの意味を伝える（断定ではなく「一般的に〜と言われています」調）。
    - 参照：内閣府「令和7年版高齢社会白書」（https://www8.cao.go.jp/kourei/whitepaper/w-2025/html/zenbun/s1_1_1.html）— 高齢化率・健康寿命に関する一般的背景データとして言及。
  - H3：50代は「自分の備え」と「親の備え」が重なる世代
    - 50代は自分自身の生前整理を意識し始める一方で、親の介護・終活サポートという役割を同時に担うことが多い（いわゆる「サンドイッチ世代」）。自分が率先して生前整理を始めておくと、親との終活話の糸口にもなりやすい。この「二重の当事者性」こそ50代固有の動機。
    - 独自切り口スロット（後述）に該当。
  - H3：「時間があるから少しずつ」が最大のメリット
    - 70代・80代になって急いで始めるより、50代の今から少しずつ進めれば焦りがない。1日1引き出し、月1箱など「マイクロタスク」で10年かけて仕上げる発想がむしろ豊かな時間の使い方。
- **公的出典候補**：内閣府「令和7年版高齢社会白書」（https://www8.cao.go.jp/kourei/whitepaper/w-2025/html/zenbun/s1_1_1.html）、総務省「人口推計2024年10月1日現在」（https://www.soumu.go.jp/menu_news/s-news/01toukei03_01000130.html）
- 【フクロウ相槌スロット】：「50代って、ちょうどいいスタートラインにいるんだホー。急がなくていい、でも今が動き時だホー。」

### 3. H2：50代の生前整理でやること——4つの柱
- **狙い**：検索意図(3)を回収。「やることリスト」の要求に応えながら、深掘りしすぎず50代向けに絞り込んだ優先度を示す。C10-040（生前整理 進め方）との棲み分けは「50代視点」でのフィルタリング。法務NG（個別税額・節税・遺言文案）を徹底回避。
- **骨子**：
  - H3：(1) 身の回りのものを仕分ける——「使うか・使わないか」から始める
    - 衣類・書籍・家電・思い出の品を「使う/使わない/保留」の3分類で進める。50代は子どもの独立で空き部屋が生まれるタイミングでもあり、ライフスタイルの変化を整理のきっかけにしやすい。一気にやらず「今日はこの引き出し一段だけ」のマイクロタスク推奨。
  - H3：(2) 書類・財産の保管場所を一覧化する
    - 通帳・保険証券・権利証・年金手帳など重要書類がどこにあるかを家族が把握できる状態にしておく。「財産一覧表」を作る目的は遺族が困らないようにするためであり、**具体的な税額計算・相続手続きの代行は専門家（税理士・司法書士）へ**。
    - 出典候補：自治体・法務省の公的案内へのリンクを一般論で促す程度にとどめる。
  - H3：(3) デジタルの棚卸しをする
    - ネット銀行・証券口座・電子マネー・サブスクリプションのリスト化。国民生活センター（2024年11月発表「デジタル終活」）によると、ID・パスワード情報がなく遺族がサブスク解約できないトラブルが増加。「まず何があるか書き出す」一歩から。
    - 参照：国民生活センター「今から考えておきたい『デジタル終活』」（https://www.kokusen.go.jp/news/data/n-20241120_1.html）
  - H3：(4) エンディングノートを書き始める
    - エンディングノートは遺言書と異なり法的拘束力はなく、家族への希望・連絡先・医療希望などを自由に書けるもの。「まず1ページだけ」から始めるハードルの低さを伝える。書き方の詳細は内部リンク（/guidebook）へ。
    - 遺言書の具体的文案・法的効力の詳細は専門家（弁護士）へ（非弁遵守）。
- **公的出典候補**：国民生活センター「今から考えておきたい『デジタル終活』」（https://www.kokusen.go.jp/news/data/n-20241120_1.html）、国民生活センター「生前整理 デジタル遺品リストを作りましょう」（https://www.kokusen.go.jp/mimamori/mj_mailmag/mj-shinsen430.html）

### 4. H2：無理なく続けるための3つのコツ
- **狙い**：検索意図(4)を回収。「始めたが続かない」「一気にやろうとして挫折」という生前整理あるある障壁を取り除く。table stakes であり上位の弱点は「コツの抽象度が高い」点。具体的な行動単位まで落とし込む。
- **骨子**：
  - コツ(1)：今日は「この場所だけ」と決める——マイクロタスク化
    - 引き出し一段・押し入れの上段だけ・財布の中の領収書だけ、など単位を小さく設定。完了感が次への動力になる。
  - コツ(2)：迷ったものは「保留ボックス」に入れて後で判断する
    - 思い出の品で手が止まる→保留ボックスに一時保管し、半年後に改めて判断するルールを作る。感情的に決断するより後から見直した方が納得感が高い。
  - コツ(3)：家族に「今やっていること」を伝えておく
    - 一人で抱え込まず、家族に整理の進捗を共有しておくと、万が一のときに「どこに何があるか」が伝わる。親との終活話をするきっかけにもなる。
- **公的出典候補**：特になし（実践的なコツのため一般論で構成）
